### 1.2 目录职责说明

| 目录 | 职责 | 关键内容 |
|------|------|---------|
| types/ | 类型定义 | 实体、状态、接口类型 |
| supports/ | 支持性内容 | API、参数、错误、常量 |
| foundations/ | 底层能力 | 适配器、客户端、存储、编排 |
| features/ | 功能特性 | Slices、Commands、Actors、Selectors |
| hooks/ | React Hooks | UI 层使用的 Hooks |
| application/ | 应用层 | 模块初始化、引导、依赖注入 |

### 1.3 文件命名规范

**类型文件**：
- 实体类型：小写驼峰，如 `identity.ts`
- 状态类型：小写驼峰 + State 后缀，如 `terminalIdentityState.ts`
- 接口类型：大写 I 前缀，如 `IDeviceInfoAdapter.ts`

**功能文件**：
- Slice：小写驼峰 + Slice 后缀，如 `terminalIdentitySlice.ts`
- Command：小写驼峰 + Commands 后缀，如 `identityCommands.ts`
- Actor：大写驼峰 + Actor 后缀，如 `IdentityActor.ts`
- Selector：小写驼峰 + Selectors 后缀，如 `identitySelectors.ts`

**Foundation 文件**：
- Adapter：大写驼峰 + Adapter 后缀，如 `DeviceInfoAdapter.ts`
- Client：大写驼峰 + Client 后缀，如 `TcpControlClient.ts`
- Repository：大写驼峰 + Repository 后缀，如 `CredentialRepository.ts`
- Coordinator：大写驼峰 + Coordinator 后缀，如 `SessionCoordinator.ts`

## 4. modulePreSetup 设计

### 4.1 职责定位

`modulePreSetup` 是模块初始化的关键函数，在模块加载时执行，负责：

**核心职责**：
1. 注册 Foundation Adapters
2. 初始化 Coordinators
3. 加载本地缓存
4. 注册请求拦截器
5. 注册生命周期监听器

**不应承担的职责**：
- ❌ 直接发起业务请求
- ❌ 直接构造 UI 逻辑
- ❌ 直接依赖项目级远控行为

### 4.2 完整实现

```typescript
// src/application/modulePreSetup.ts
import type { ModulePreSetupContext } from '@impos2/kernel-core-base'
import { terminalIdentitySlice } from '../features/slices/terminalIdentitySlice'
import { terminalBindingSlice } from '../features/slices/terminalBindingSlice'
import { terminalDataCommands } from '../features/commands'
import {
  registerDeviceInfoAdapter,
  registerStorageAdapter,
  registerTcpTransportAdapter,
  registerTdpTransportAdapter,
  registerNetworkStateAdapter,
  registerAppLifecycleAdapter,
  registerLoggerAdapter,
  registerClockAdapter,
} from '../foundations/adapters'
import {
  sessionCoordinator,
  projectionSyncCoordinator,
} from '../foundations/coordinators'
import { credentialRepository } from '../foundations/repositories'
import { selectTerminalCredential } from '../features/selectors'

export const terminalDataModulePreSetup = async (
  context: ModulePreSetupContext
) => {
  const { dispatch, getState, adapters } = context

  // 1. 注册 Adapters
  registerDeviceInfoAdapter(adapters.deviceInfo)
  registerStorageAdapter(adapters.storage)
  registerTcpTransportAdapter(adapters.tcpTransport)
  registerTdpTransportAdapter(adapters.tdpTransport)
  registerNetworkStateAdapter(adapters.networkState)
  registerAppLifecycleAdapter(adapters.appLifecycle)
  registerLoggerAdapter(adapters.logger)
  registerClockAdapter(adapters.clock)

  // 2. 初始化 Coordinators
  await sessionCoordinator.initialize()
  await projectionSyncCoordinator.initialize()

  // 3. 加载本地缓存
  const cachedIdentity = await credentialRepository.load()
  if (cachedIdentity) {
    dispatch(
      terminalIdentitySlice.actions.setIdentity({
        terminalId: cachedIdentity.terminalId,
        deviceId: cachedIdentity.deviceId,
        deviceFingerprint: cachedIdentity.deviceFingerprint,
        activationStatus: cachedIdentity.activationStatus,
        activatedAt: cachedIdentity.activatedAt,
        lastActivatedAt: cachedIdentity.lastActivatedAt,
      })
    )
    dispatch(
      terminalIdentitySlice.actions.setCredential(cachedIdentity.credential)
    )
  }

  // 4. 注册 TCP Token Interceptor
  adapters.tcpTransport.registerRequestInterceptor(async (config) => {
    const state = getState()
    const credential = selectTerminalCredential(state)
    if (credential?.accessToken) {
      config.headers = config.headers || {}
      config.headers.Authorization = `Bearer ${credential.accessToken}`
    }
    return config
  })

  // 5. 注册生命周期监听器
  adapters.appLifecycle.onAppActive(() => {
    dispatch(terminalDataCommands.reconnectTdpSession())
  })

  adapters.appLifecycle.onAppBackground(() => {
    dispatch(terminalDataCommands.disconnectTdpSession())
  })

  // 6. 注册网络状态监听器
  adapters.networkState.onNetworkAvailable(() => {
    dispatch(terminalDataCommands.reconnectTdpSession())
  })

  adapters.networkState.onNetworkUnavailable(() => {
    dispatch(terminalDataCommands.disconnectTdpSession())
  })
}
```

### 4.3 执行时机

```
应用启动
    ↓
加载 AppModule
    ↓
执行 modulePreSetup  ← 在这里初始化
    ↓
注册 Slices/Commands/Actors
    ↓
应用就绪
```

### 4.4 错误处理

```typescript
export const terminalDataModulePreSetup = async (
  context: ModulePreSetupContext
) => {
  try {
    // 初始化逻辑
    // ...
  } catch (error) {
    context.logger.error('Terminal data module pre-setup failed', error)
    // 不抛出错误，允许应用继续启动
    // 但记录错误状态
    context.dispatch(
      terminalIdentitySlice.actions.setLastError(
        error instanceof Error ? error.message : 'Unknown error'
      )
    )
  }
}
```

#### 5.1.3 为什么 TDP 不全部使用 HTTP

**原因**：

1. **实时性要求**
   - WebSocket 提供双向实时通信
   - 服务端可以主动推送数据
   - 减少轮询开销

2. **协议灵活性**
   - 未来可能支持 SSE（Server-Sent Events）
   - 未来可能支持 HTTP Long Polling
   - 未来可能支持原生推送通道

3. **架构设计**
   - `supports/apis` 提供默认协议描述
   - `foundations/adapters` 提供 transport 抽象
   - 上层代码不感知具体传输方式

**抽象示例**：

## 6. Foundations 设计

### 6.1 Adapters（适配器）

#### 6.1.1 设计理念

**为什么需要 Adapter**：
- 抽象平台差异（RN / Electron / Web）
- 便于单元测试（可 Mock）
- 支持多平台复用
- 降低核心逻辑与平台的耦合

**Adapter 设计原则**：
- 接口定义在 `terminal-data` 包中
- 实现由适配层注入
- 核心逻辑只依赖接口，不依赖实现

```typescript
// src/foundations/clients/TdpSessionClient.ts
import type { ITdpTransportAdapter, TdpMessage } from '../adapters/ITdpTransportAdapter'
import type { TdpSessionDescriptor } from '../../types'

export class TdpSessionClient {
  constructor(private transport: ITdpTransportAdapter) {}

  /**
   * 连接会话
   */
  async connect(params: {
    terminalId: string
    accessToken: string
  }): Promise<TdpSessionDescriptor> {
    const url = this.buildSessionUrl(params)
    await this.transport.connect(url)

    // 等待服务端返回 SESSION_CONNECTED 消息
    return new Promise((resolve, reject) => {
      const timeout = setTimeout(() => {
        reject(new Error('Session connect timeout'))
      }, 10000)

      this.transport.onMessage((message) => {
        if (message.type === 'SESSION_CONNECTED') {
          clearTimeout(timeout)
          resolve(message.payload as TdpSessionDescriptor)
        }
      })
    })
  }

  /**
   * 发送心跳
   */
  async heartbeat(): Promise<void> {
    await this.transport.send({
      type: 'HEARTBEAT',
      payload: { timestamp: Date.now() },
      timestamp: Date.now(),
    })
  }

  /**
   * 断开连接
   */
  async disconnect(): Promise<void> {
    await this.transport.disconnect()
  }

  /**
   * 监听消息
   */
  onMessage(handler: (message: TdpMessage) => void): void {
    this.transport.onMessage(handler)
  }

  private buildSessionUrl(params: { terminalId: string; accessToken: string }): string {
    const baseUrl = process.env.TDP_WS_URL || 'ws://localhost:3001'
    return `${baseUrl}/api/tdp/v1/session?terminalId=${params.terminalId}&token=${params.accessToken}`
  }
}
```

```typescript
import type { IStorageAdapter } from '../adapters/IStorageAdapter'
import type { TerminalIdentity, TerminalCredential } from '../../types'

const STORAGE_KEY_IDENTITY = 'terminal-data:identity'
const STORAGE_KEY_CREDENTIAL = 'terminal-data:credential'

export class CredentialRepository {
  constructor(private storage: IStorageAdapter) {}

  /**
   * 保存身份和凭证
   */
  async save(data: {
    identity: TerminalIdentity
    credential: TerminalCredential
  }): Promise<void> {
    await this.storage.multiSet([
      [STORAGE_KEY_IDENTITY, JSON.stringify(data.identity)],
      [STORAGE_KEY_CREDENTIAL, JSON.stringify(data.credential)],
    ])
  }

  /**
   * 加载身份和凭证
   */
  async load(): Promise<{
    identity: TerminalIdentity
    credential: TerminalCredential
  } | null> {
    const results = await this.storage.multiGet([
      STORAGE_KEY_IDENTITY,
      STORAGE_KEY_CREDENTIAL,
    ])

    const identityStr = results[0][1]
    const credentialStr = results[1][1]

    if (!identityStr || !credentialStr) {
      return null
    }

    return {
      identity: JSON.parse(identityStr),
      credential: JSON.parse(credentialStr),
    }
  }

  /**
   * 清除身份和凭证
   */
  async clear(): Promise<void> {
    await this.storage.multiRemove([STORAGE_KEY_IDENTITY, STORAGE_KEY_CREDENTIAL])
  }
}
```

```typescript
// src/foundations/repositories/ProjectionRepository.ts
import type { IStorageAdapter } from '../adapters/IStorageAdapter'
import type { ProjectionRecord } from '../../types'

const STORAGE_KEY_PREFIX = 'terminal-data:projection:'

export class ProjectionRepository {
  constructor(private storage: IStorageAdapter) {}

  /**
   * 保存 Projection
   */
  async save(projection: ProjectionRecord): Promise<void> {
    const key = this.buildKey(projection.topicKey, projection.scopeKey)
    await this.storage.set(key, JSON.stringify(projection))
  }

  /**
   * 加载 Projection
   */
  async load(topicKey: string, scopeKey: string): Promise<ProjectionRecord | null> {
    const key = this.buildKey(topicKey, scopeKey)
    const data = await this.storage.get(key)
    return data ? JSON.parse(data) : null
  }

  /**
   * 删除 Projection
   */
  async remove(topicKey: string, scopeKey: string): Promise<void> {
    const key = this.buildKey(topicKey, scopeKey)
    await this.storage.remove(key)
  }

  /**
   * 加载所有 Projections
   */
  async loadAll(): Promise<ProjectionRecord[]> {
    const allKeys = await this.storage.getAllKeys()
    const projectionKeys = allKeys.filter(key => key.startsWith(STORAGE_KEY_PREFIX))

    if (projectionKeys.length === 0) {
      return []
    }

    const results = await this.storage.multiGet(projectionKeys)
    return results
      .map(([_, value]) => (value ? JSON.parse(value) : null))
      .filter(Boolean)
  }

  private buildKey(topicKey: string, scopeKey: string): string {
    return `${STORAGE_KEY_PREFIX}${topicKey}:${scopeKey}`
  }
}
```

#### 6.4.1 SessionCoordinator

```typescript
// src/foundations/coordinators/SessionCoordinator.ts
import type { TdpSessionClient } from '../clients/TdpSessionClient'
import type { ILoggerAdapter } from '../adapters/ILoggerAdapter'
import { tdpParameters } from '../../supports/parameters/tdpParameters'

export class SessionCoordinator {
  private heartbeatTimer?: number
  private reconnectAttempts = 0

  constructor(
    private sessionClient: TdpSessionClient,
    private logger: ILoggerAdapter
  ) {}

  async initialize(): Promise<void> {
    this.logger.info('SessionCoordinator initialized')
  }

  /**
   * 启动心跳
   */
  startHeartbeat(): void {
    this.stopHeartbeat()

    this.heartbeatTimer = setInterval(() => {
      this.sessionClient.heartbeat().catch((error) => {
        this.logger.error('Heartbeat failed', error)
      })
    }, tdpParameters.heartbeatInterval)
  }

  /**
   * 停止心跳
   */
  stopHeartbeat(): void {
    if (this.heartbeatTimer) {
      clearInterval(this.heartbeatTimer)
      this.heartbeatTimer = undefined
    }
  }

  /**
   * 重连
   */
  async reconnect(params: { terminalId: string; accessToken: string }): Promise<void> {
    if (this.reconnectAttempts >= tdpParameters.maxReconnectAttempts) {
      throw new Error('Max reconnect attempts reached')
    }

    const backoff = tdpParameters.reconnectBackoff[
      Math.min(this.reconnectAttempts, tdpParameters.reconnectBackoff.length - 1)
    ]

    await new Promise(resolve => setTimeout(resolve, backoff))

    try {
      await this.sessionClient.connect(params)
      this.reconnectAttempts = 0
      this.startHeartbeat()
    } catch (error) {
      this.reconnectAttempts++
      throw error
    }
  }

  /**
   * 重置重连计数
   */
  resetReconnectAttempts(): void {
    this.reconnectAttempts = 0
  }
}
```

#### 6.4.2 ProjectionSyncCoordinator

```typescript
// src/foundations/coordinators/ProjectionSyncCoordinator.ts
import type { TdpSyncClient } from '../clients/TdpSyncClient'
import type { ProjectionRepository } from '../repositories/ProjectionRepository'
import type { ILoggerAdapter } from '../adapters/ILoggerAdapter'
import { syncParameters } from '../../supports/parameters/syncParameters'

export class ProjectionSyncCoordinator {
  private syncTimers: Map<string, number> = new Map()

  constructor(
    private syncClient: TdpSyncClient,
    private projectionRepository: ProjectionRepository,
    private logger: ILoggerAdapter
  ) {}

  async initialize(): Promise<void> {
    this.logger.info('ProjectionSyncCoordinator initialized')
  }

  /**
   * 启动 Projection 定期同步
   */
  startPeriodicSync(topicKey: string, scopeKey: string): void {
    this.stopPeriodicSync(topicKey)

    const timer = setInterval(async () => {
      try {
        await this.syncSnapshot(topicKey, scopeKey)
      } catch (error) {
        this.logger.error(`Periodic sync failed for ${topicKey}`, error)
      }
    }, syncParameters.snapshotSyncInterval)

    this.syncTimers.set(topicKey, timer)
  }

  /**
   * 停止 Projection 定期同步
   */
  stopPeriodicSync(topicKey: string): void {
    const timer = this.syncTimers.get(topicKey)
    if (timer) {
      clearInterval(timer)
      this.syncTimers.delete(topicKey)
    }
  }

  /**
   * 同步 Projection 快照
   */
  async syncSnapshot(topicKey: string, scopeKey: string): Promise<void> {
    const projection = await this.syncClient.fetchSnapshot({
      topicKey,
      scopeType: 'STORE',
      scopeKey,
    })

    await this.projectionRepository.save(projection)
    this.logger.info(`Snapshot synced for ${topicKey}:${scopeKey}`)
  }

  /**
   * 追赶 Change 增量
   */
  async catchUpChanges(
    topicKey: string,
    scopeKey: string,
    fromRevision: number
  ): Promise<void> {
    let currentRevision = fromRevision
    let hasMore = true

    while (hasMore) {
      const changes = await this.syncClient.fetchChanges({
        topicKey,
        scopeType: 'STORE',
        scopeKey,
        fromRevision: currentRevision,
        limit: syncParameters.changeCatchUpBatchSize,
      })

      if (changes.length === 0) {
        hasMore = false
        break
      }

      // 应用 changes
      for (const change of changes) {
        await this.applyChange(change)
        currentRevision = change.revision
      }

      // 如果返回的 changes 少于批次大小，说明已经追上了
      if (changes.length < syncParameters.changeCatchUpBatchSize) {
        hasMore = false
      }
    }

    this.logger.info(`Caught up changes for ${topicKey}:${scopeKey} to revision ${currentRevision}`)
  }

  /**
   * 应用单个 Change
   */
  private async applyChange(change: ChangeRecord): Promise<void> {
    const projection = await this.projectionRepository.load(
      change.topicKey,
      change.scopeKey
    )

    if (!projection) {
      // 如果本地没有 projection，需要先同步快照
      await this.syncSnapshot(change.topicKey, change.scopeKey)
      return
    }

    // 应用 change 到 projection
    const updatedProjection = {
      ...projection,
      revision: change.revision,
      payload: this.mergePayload(projection.payload, change.payload),
      updatedAt: change.timestamp,
    }

    await this.projectionRepository.save(updatedProjection)
  }

  /**
   * 合并 payload
   */
  private mergePayload(
    base: Record<string, unknown>,
    change: Record<string, unknown>
```

#### 6.4.3 ReportCoordinator

```typescript
// src/foundations/coordinators/ReportCoordinator.ts
import type { TcpControlClient } from '../clients/TcpControlClient'
import type { ILoggerAdapter } from '../adapters/ILoggerAdapter'
import { tcpParameters } from '../../supports/parameters/tcpParameters'

export class ReportCoordinator {
  private reportQueue: TerminalReportPayload[] = []
  private reportTimer?: number
  private isReporting = false

  constructor(
    private tcpClient: TcpControlClient,
    private logger: ILoggerAdapter
  ) {}

  async initialize(): Promise<void> {
    this.startPeriodicReport()
    this.logger.info('ReportCoordinator initialized')
  }

  /**
   * 启动定期上报
   */
  startPeriodicReport(): void {
    this.stopPeriodicReport()

    this.reportTimer = setInterval(() => {
      this.flushReports()
    }, tcpParameters.reportInterval)
  }

  /**
   * 停止定期上报
   */
  stopPeriodicReport(): void {
    if (this.reportTimer) {
      clearInterval(this.reportTimer)
      this.reportTimer = undefined
    }
  }

  /**
   * 添加上报到队列
   */
  enqueueReport(report: TerminalReportPayload): void {
    this.reportQueue.push(report)
    this.logger.debug(`Report enqueued: ${report.reportType}`)
  }

  /**
   * 立即上报（不等待定期上报）
   */
  async reportImmediately(
    terminalId: string,
    report: TerminalReportPayload
  ): Promise<void> {
    try {
      await this.tcpClient.reportTaskResult(terminalId, report)
      this.logger.info(`Report sent immediately: ${report.reportType}`)
    } catch (error) {
      this.logger.error('Immediate report failed', error)
      // 失败后加入队列，等待下次上报
      this.enqueueReport(report)
    }
  }

  /**
   * 刷新上报队列
   */
  private async flushReports(): Promise<void> {
    if (this.isReporting || this.reportQueue.length === 0) {
      return
    }

    this.isReporting = true

    try {
      const reports = [...this.reportQueue]
      this.reportQueue = []

      for (const report of reports) {
        try {
          // 这里需要从 state 中获取 terminalId
          // 实际实现中应该通过依赖注入或其他方式获取
          // await this.tcpClient.reportTaskResult(terminalId, report)
          this.logger.info(`Report sent: ${report.reportType}`)
        } catch (error) {
          this.logger.error('Report failed', error)
          // 失败的报告重新加入队列
          this.reportQueue.push(report)
        }
      }
    } finally {
      this.isReporting = false
    }
  }
}
```

```typescript
export class DerivedTreeBuilder {
  /**

  buildFromBindingContext(context: TerminalBindingContext): DerivedTreeNode[] {
    const nodes: DerivedTreeNode[] = []

    // 创建 ROOT 节点

      refId: context.tenant.tenantId,
      label: context.tenant.name,
      parentNodeId: 'root',
      childrenNodeIds: [context.brand.brandId],
      metadata: {
        code: context.tenant.code,
      },
    })

    // 创建 BRAND 节点
    nodes.push({
      nodeId: context.brand.brandId,
      nodeType: 'BRAND' as DerivedTreeNodeType,
      refId: context.brand.brandId,
      label: context.brand.name,
      parentNodeId: context.tenant.tenantId,
      childrenNodeIds: [context.store.storeId],
      metadata: {
        code: context.brand.code,
      },
    })

    // 创建 STORE 节点
    nodes.push({
      nodeId: context.store.storeId,
      nodeType: 'STORE' as DerivedTreeNodeType,
      refId: context.store.storeId,
      label: context.store.name,
      parentNodeId: context.brand.brandId,
      childrenNodeIds: [context.terminalId],
      metadata: {
        code: context.store.code,
      },
    })

    // 创建 TERMINAL 节点
    nodes.push({
      nodeId: context.terminalId,
      nodeType: 'TERMINAL' as DerivedTreeNodeType,
      refId: context.terminalId,
      label: `Terminal ${context.terminalId}`,
      parentNodeId: context.store.storeId,
      childrenNodeIds: [context.profile.profileId],
    })

    // 创建 PROFILE 节点
    nodes.push({
      nodeId: context.profile.profileId,
      nodeType: 'PROFILE' as DerivedTreeNodeType,
      refId: context.profile.profileId,
      label: context.profile.name,
      parentNodeId: context.terminalId,
      childrenNodeIds: [],
      metadata: {
        code: context.profile.code,
      },
    })

    return nodes
  }

  /**
   * 查找节点
   */
  findNode(nodes: DerivedTreeNode[], nodeId: string): DerivedTreeNode | undefined {
    return nodes.find(node => node.nodeId === nodeId)
  }

  /**
   * 查找节点的所有子节点
   */
  findChildren(nodes: DerivedTreeNode[], parentNodeId: string): DerivedTreeNode[] {
    return nodes.filter(node => node.parentNodeId === parentNodeId)
  }

  /**
   * 查找节点的所有祖先节点
   */
  findAncestors(nodes: DerivedTreeNode[], nodeId: string): DerivedTreeNode[] {
    const ancestors: DerivedTreeNode[] = []
    let currentNode = this.findNode(nodes, nodeId)

    while (currentNode && currentNode.parentNodeId) {
      const parent = this.findNode(nodes, currentNode.parentNodeId)
      if (parent) {
        ancestors.push(parent)
        currentNode = parent
      } else {
        break
      }
    }

    return ancestors
  }
}
```

```typescript
// src/foundations/resolvers/ScopeResolver.ts
import type { TerminalBindingContext } from '../../types'
import type { TdpScopeType } from '../../types'

export class ScopeResolver {
  /**
   * 解析 Scope Key
   */
  resolveScopeKey(
    context: TerminalBindingContext,
    scopeType: TdpScopeType
  ): string {
    switch (scopeType) {
      case 'TENANT':
        return context.tenant.tenantId
      case 'BRAND':
        return context.brand.brandId
      case 'STORE':
        return context.store.storeId
      case 'TERMINAL':
        return context.terminalId
      case 'PROFILE':
        return context.profile.profileId
      default:
        throw new Error(`Unknown scope type: ${scopeType}`)
    }
  }

  /**
   * 判断 Scope 是否匹配
   */
  isScopeMatch(
    context: TerminalBindingContext,
    scopeType: TdpScopeType,
    scopeKey: string
  ): boolean {
    const resolvedKey = this.resolveScopeKey(context, scopeType)
    return resolvedKey === scopeKey
  }

  /**
   * 获取 Scope 层级
   */
  getScopeLevel(scopeType: TdpScopeType): number {
    const levels: Record<TdpScopeType, number> = {
      TENANT: 1,
      BRAND: 2,
      STORE: 3,
      TERMINAL: 4,
      PROFILE: 5,
    }
    return levels[scopeType] || 0
  }

  /**
   * 判断 Scope 是否包含另一个 Scope
   */
  isScopeContains(
    context: TerminalBindingContext,
    parentScopeType: TdpScopeType,
    parentScopeKey: string,
    childScopeType: TdpScopeType,
    childScopeKey: string
  ): boolean {
    // 父 Scope 层级必须小于子 Scope 层级
    if (this.getScopeLevel(parentScopeType) >= this.getScopeLevel(childScopeType)) {
      return false
    }

    // 检查父 Scope 是否匹配
    if (!this.isScopeMatch(context, parentScopeType, parentScopeKey)) {
      return false
    }

    // 检查子 Scope 是否匹配
    return this.isScopeMatch(context, childScopeType, childScopeKey)
  }
}
```

```typescript
import { terminalDerivedTreeSlice } from '../slices/terminalDerivedTreeSlice'
import { derivedTreeBuilder } from '../../foundations/builders'

export const rebuildDerivedTreeActor = createActor({
  command: terminalDataCommands.rebuildDerivedTree,
  execute: async ({ dispatch, getState }) => {
    const state = getState()
    const bindingContext = state.terminalBinding.bindingContext

    if (!bindingContext) {
      throw new Error('No binding context')
    }

    // 构建派生树
    const nodes = derivedTreeBuilder.buildFromBindingContext(bindingContext)

    // 更新状态
    dispatch(terminalDerivedTreeSlice.actions.setNodes(nodes))
    dispatch(terminalDerivedTreeSlice.actions.setLastBuiltAt(Date.now()))
  },
})
```

## 9. 总结

### 9.1 模块结构总结

本文档详细阐述了 `terminal-data` 包的完整模块结构：

**五层架构**：
1. **types**：类型定义层
2. **supports**：支持性内容层（APIs、Parameters、Errors、Constants）
3. **foundations**：底层能力层（Adapters、Clients、Repositories、Coordinators、Builders、Resolvers）
4. **features**：功能特性层（Slices、Commands、Actors、Selectors）
5. **hooks**：React Hooks 层

### 9.2 关键设计原则

1. **状态权威在 Slices**
   - 所有状态必须在 Redux slices 中定义
   - 不允许状态散落在单例或全局变量中

2. **动作抽象在 Commands**
   - 所有关键动作使用 `createModuleCommands` 统一建模
   - 避免上层直接调用散落函数

3. **副作用执行在 Actors**
   - 所有副作用由 Actor 承担编排
   - 包括 HTTP 调用、Session 建立、数据同步、本地持久化

4. **底层抽象在 Foundations**
   - 平台差异通过 Adapter 抽象
   - 协议细节通过 Client 封装
   - 本地存储通过 Repository 抽象
   - 流程编排通过 Coordinator 实现

5. **UI 集成通过 Hooks**
   - UI 层通过 Hooks 访问状态
   - UI 层通过 Commands 驱动动作
   - 保持 UI 层与核心逻辑解耦

### 9.3 导出规范

**必须导出**：
- ✅ 模块定义（`terminalDataModule`）
- ✅ 所有 Slices
- ✅ 所有 Commands
- ✅ 所有 Selectors
- ✅ 所有 Hooks
- ✅ 所有类型定义
- ✅ API、参数、错误、常量定义
- ✅ Foundation 接口类型

**不应导出**：
- ❌ Foundation 实现类（由适配层注入）
- ❌ 内部辅助函数
- ❌ 测试代码
- ❌ Mock 数据

### 9.4 与 1-kernel 框架对齐

- 遵循 `AppModule` 结构
- 使用 `createModuleCommands` 定义命令
- 使用 `createActor` 定义 Actor
- 使用 `createSlice` 定义状态
- 通过 `modulePreSetup` 初始化模块
- 通过 `dependencies` 声明依赖

### 9.5 文件组织原则

- **高内聚、低耦合**：每个文件职责单一
- **小文件优于大文件**：200-400 行典型，800 行最大
- **按功能组织**：按子域划分，不按类型划分
- **清晰的导出**：每个目录有 `index.ts` 统一导出
