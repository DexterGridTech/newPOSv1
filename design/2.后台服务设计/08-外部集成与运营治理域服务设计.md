# 08-外部集成与运营治理域服务设计

> **文档说明（P1-1）：** 本文档是**两个独立限界上下文微服务**的合集，仅为节省篇幅合并成一份文档，**不代表两者合并为一个服务**。
> - `external-service`（外部集成域服务）：独立部署，端口 8087，数据库 external_db
> - `governance-service`（运营治理域服务）：独立部署，端口 8088，数据库 governance_db
>
> 两者职责、团队分工、部署资源、权限边界完全独立，请勿混淆。

## 1. 服务概述

本文档涵盖外部集成域服务（external-service）和运营治理域服务（governance-service）两个服务的完整设计。

### 1.1 外部集成域服务

外部集成域服务（external-service）负责与外部系统的集成，包括会员系统、配送平台、ERP系统、支付网关等。

**核心职责**：
- 会员系统集成
- 配送平台集成
- ERP系统集成
- 支付网关集成
- 外部调用日志记录

**核心聚合根**：
- MemberIntegration（会员集成）
- DeliveryPlatformIntegration（配送平台集成）
- ERPIntegration（ERP集成）

### 1.2 运营治理域服务

运营治理域服务（governance-service）负责异常处理、补偿任务、审计日志、运营监控等功能。

**核心职责**：
- 异常台账管理
- 补偿任务执行
- 审计日志记录
- 运营指标监控
- 告警规则管理

**核心聚合根**：
- ExceptionRecord（异常记录）
- CompensationTask（补偿任务）
- AuditLog（审计日志）
- AlertRule（告警规则）

```java
@Service
public class MemberIntegrationService {

    @Autowired
    private MemberIntegrationRepository integrationRepository;

    @Autowired
    private ExternalCallLogRepository callLogRepository;

    /**
     * 验证会员
     */
    public MemberInfo verifyMember(String mallId, String memberId) {
        MemberIntegration integration = integrationRepository.findByMallId(mallId)
            .orElseThrow(() -> new MemberIntegrationNotFoundException(mallId));

        try {
            long startTime = System.currentTimeMillis();

            // 调用会员系统API
            MemberApiResponse response = callMemberApi(
                integration,
                "/member/verify",
                Map.of("memberId", memberId)
            );

            long duration = System.currentTimeMillis() - startTime;

            // 记录调用日志
            recordCallLog(integration, "VERIFY_MEMBER", memberId, response, duration, true);

            return MemberInfo.builder()
                .memberId(response.getMemberId())
                .memberName(response.getMemberName())
                .memberLevel(response.getMemberLevel())
                .availablePoints(response.getAvailablePoints())
                .isActive(response.getIsActive())
                .build();

        } catch (Exception e) {
            recordCallLog(integration, "VERIFY_MEMBER", memberId, null, 0, false);
            throw new MemberIntegrationException("会员验证失败", e);
        }
    }

    /**
     * 锁定优惠券
     */
    public LockCouponResult lockCoupon(String mallId, String couponId, String orderId) {
        MemberIntegration integration = integrationRepository.findByMallId(mallId)
            .orElseThrow(() -> new MemberIntegrationNotFoundException(mallId));

        try {
            Map<String, Object> params = Map.of(
                "couponId", couponId,
                "orderId", orderId
            );

            MemberApiResponse response = callMemberApi(
                integration,
                "/coupon/lock",
                params
            );

            return LockCouponResult.builder()
                .lockId(response.getLockId())
                .lockedAt(response.getLockedAt())
                .expireAt(response.getExpireAt())
                .build();

        } catch (Exception e) {
            throw new MemberIntegrationException("优惠券锁定失败", e);
        }
    }

    /**
     * 核销优惠券
     */
    public RedeemCouponResult redeemCoupon(String mallId, String couponId, String orderId) {
        MemberIntegration integration = integrationRepository.findByMallId(mallId)
            .orElseThrow(() -> new MemberIntegrationNotFoundException(mallId));

        try {
            Map<String, Object> params = Map.of(
                "couponId", couponId,
                "orderId", orderId
            );

            MemberApiResponse response = callMemberApi(
                integration,
                "/coupon/redeem",
                params
            );

            return RedeemCouponResult.builder()
                .redemptionId(response.getRedemptionId())
                .redeemedAt(response.getRedeemedAt())
                .discountAmount(response.getDiscountAmount())
                .build();

        } catch (Exception e) {
            throw new MemberIntegrationException("优惠券核销失败", e);
        }
    }

    private MemberApiResponse callMemberApi(
        MemberIntegration integration,
        String path,
        Map<String, Object> params
    ) {
        String url = integration.getApiEndpoint() + path;

        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);
        headers.set("Authorization", "Bearer " + integration.getApiKey());

        HttpEntity<Map<String, Object>> request = new HttpEntity<>(params, headers);

        ResponseEntity<MemberApiResponse> response = restTemplate.postForEntity(
            url,
            request,
            MemberApiResponse.class
        );

        return response.getBody();
    }

    private void recordCallLog(
        MemberIntegration integration,
        String operationType,
        Object request,
        Object response,
        long duration,
        boolean success
    ) {
        ExternalCallLog log = ExternalCallLog.builder()
            .logId(IdGenerator.generateLogId())
            .integrationId(integration.getIntegrationId())
            .integrationType(IntegrationType.MEMBER)
            .operationType(operationType)
            .requestPayload(JSON.toJSONString(request))
            .responsePayload(response != null ? JSON.toJSONString(response) : null)
            .durationMs(duration)
            .executionStatus(success ? ExecutionStatus.SUCCESS : ExecutionStatus.FAILED)
            .createdAt(LocalDateTime.now())
            .build();

        callLogRepository.save(log);
    }
}
```

```java
@Service
public class DeliveryIntegrationService {

    @Autowired
    private Map<String, DeliveryPlatformAdapter> adapters;

    /**
     * 创建配送订单
     */
    public DeliveryOrderResult createDeliveryOrder(CreateDeliveryOrderRequest request) {
        // 选择配送平台
        String platformType = selectDeliveryPlatform(request);

        DeliveryPlatformAdapter adapter = adapters.get(platformType);
        if (adapter == null) {
            throw new UnsupportedDeliveryPlatformException(platformType);
        }

        try {
            return adapter.createDeliveryOrder(request);
        } catch (Exception e) {
            throw new DeliveryIntegrationException("创建配送订单失败", e);
        }
    }

    /**
     * 取消配送订单
     */
    public void cancelDeliveryOrder(String deliveryOrderId, String reason) {
        // 通过履约域提供的查询接口获取配送任务信息
        DeliveryTaskDTO task = fulfillmentQueryService.getDeliveryTask(deliveryOrderId);
        if (task == null) {
            throw new DeliveryTaskNotFoundException(deliveryOrderId);
        }

        DeliveryPlatformAdapter adapter = adapters.get(task.getDeliveryProvider());

        try {
            adapter.cancelDeliveryOrder(task.getExternalDeliveryId(), reason);
        } catch (Exception e) {
            throw new DeliveryIntegrationException("取消配送订单失败", e);
        }
    }

    /**
     * 查询配送状态
     */
    public DeliveryStatus queryDeliveryStatus(String deliveryOrderId) {
        // 通过履约域提供的查询接口获取配送任务信息
        DeliveryTaskDTO task = fulfillmentQueryService.getDeliveryTask(deliveryOrderId);
        if (task == null) {
            throw new DeliveryTaskNotFoundException(deliveryOrderId);
        }

        DeliveryPlatformAdapter adapter = adapters.get(task.getDeliveryProvider());

        try {
            return adapter.queryDeliveryStatus(task.getExternalDeliveryId());
        } catch (Exception e) {
            throw new DeliveryIntegrationException("查询配送状态失败", e);
        }
    }

    private String selectDeliveryPlatform(CreateDeliveryOrderRequest request) {
        // 简单策略：选择费用最低的平台
        List<String> availablePlatforms = List.of("DADA", "SHUNFENG", "FLASHEX");

        String selectedPlatform = null;
        Long lowestFee = Long.MAX_VALUE;

        for (String platform : availablePlatforms) {
            DeliveryPlatformAdapter adapter = adapters.get(platform);
            try {
                FeeEstimate estimate = adapter.estimateFee(
                    request.getPickupAddress(),
                    request.getDeliveryAddress(),
                    request.getWeight()
                );

                if (estimate.getEstimatedFee() < lowestFee) {
                    lowestFee = estimate.getEstimatedFee();
                    selectedPlatform = platform;
                }
            } catch (Exception e) {
                log.warn("平台 {} 费用预估失败", platform, e);
            }
        }

        return selectedPlatform != null ? selectedPlatform : "DADA";
    }
}
```

```java
@Service
public class ERPIntegrationService {

    @Autowired
    private ERPIntegrationRepository integrationRepository;

    /**
     * 同步订单到ERP
     */
    public SyncResult syncOrderToERP(String orderId) {
        SaleOrder order = orderRepository.findById(orderId)
            .orElseThrow(() -> new OrderNotFoundException(orderId));

        ERPIntegration integration = integrationRepository
            .findByTenantId(order.getTenantId())
            .orElseThrow(() -> new ERPIntegrationNotFoundException(order.getTenantId()));

        if (!integration.getSyncConfig().getOrderSyncEnabled()) {
            return SyncResult.skipped("订单同步未启用");
        }

        try {
            // 转换为ERP格式
            ERPOrderDTO erpOrder = convertToERPOrder(order, integration);

            // 调用ERP API
            ERPApiResponse response = callERPApi(
                integration,
                "/order/create",
                erpOrder
            );

            return SyncResult.success(response.getErpOrderId());

        } catch (Exception e) {
            log.error("同步订单到ERP失败: orderId={}", orderId, e);
            return SyncResult.failure(e.getMessage());
        }
    }

    /**
     * 从ERP同步库存
     */
    public SyncResult syncInventoryFromERP(String brandId) {
        ERPIntegration integration = integrationRepository
            .findByBrandId(brandId)
            .orElseThrow(() -> new ERPIntegrationNotFoundException(brandId));

        if (!integration.getSyncConfig().getInventorySyncEnabled()) {
            return SyncResult.skipped("库存同步未启用");
        }

        try {
            // 调用ERP API查询库存
            ERPApiResponse response = callERPApi(
                integration,
                "/inventory/query",
                Map.of("brandId", brandId)
            );

            List<InventoryDTO> inventories = response.getInventories();

            // 更新商品库存
            for (InventoryDTO inventory : inventories) {
                productService.updateInventory(
                    inventory.getProductCode(),
                    inventory.getAvailableStock()
                );
            }

            return SyncResult.success(inventories.size() + " 个商品库存已同步");

        } catch (Exception e) {
            log.error("从ERP同步库存失败: brandId={}", brandId, e);
            return SyncResult.failure(e.getMessage());
        }
    }

    private ERPOrderDTO convertToERPOrder(SaleOrder order, ERPIntegration integration) {
        FieldMapping mapping = integration.getFieldMapping().getOrderMapping();

        return ERPOrderDTO.builder()
            .orderNo(order.getOrderNo())
            .orderDate(order.getCreatedAt())
            .customerName(order.getCustomerName())
            .totalAmount(order.getTotalAmount() / 100.0) // 分转元
            .items(order.getItems().stream()
                .map(item -> ERPOrderItemDTO.builder()
                    .productCode(item.getProductSnapshot().getProductCode())
                    .quantity(item.getQuantity())
                    .unitPrice(item.getUnitPrice() / 100.0)
                    .build())
                .collect(Collectors.toList()))
            .build();
    }

    private ERPApiResponse callERPApi(
        ERPIntegration integration,
        String path,
        Object request
    ) {
        String url = integration.getApiEndpoint() + path;

        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);
        // 添加认证信息
        addAuthHeaders(headers, integration);

        HttpEntity<Object> httpRequest = new HttpEntity<>(request, headers);

        ResponseEntity<ERPApiResponse> response = restTemplate.postForEntity(
            url,
            httpRequest,
            ERPApiResponse.class
        );

        return response.getBody();
    }

    private void addAuthHeaders(HttpHeaders headers, ERPIntegration integration) {
        // 根据ERP系统类型添加认证信息
        switch (integration.getErpSystemType()) {
            case "KINGDEE":
                headers.set("Authorization", "Bearer " + integration.getApiKey());
                break;
            case "YONYOU":
                headers.set("X-API-Key", integration.getApiKey());
                break;
            default:
                headers.set("Authorization", integration.getApiKey());
        }
    }
}
```

```java
@Service
public class SpiInvokeService {

    @Autowired
    private MemberIntegrationRepository integrationRepository;

    @Autowired
    private SpiInvokeLogRepository logRepository;

    @Autowired
    private IdempotencyService idempotencyService;

    /**
     * 会员识别SPI调用（超时300ms，降级允许匿名）
     */
    public MemberIdentifyResult identifyMember(String storeId, String memberCode) {
        MemberIntegration integration = integrationRepository.findByStoreId(storeId)
            .orElseThrow(() -> new NotFoundException("未配置会员集成"));

        String idempotencyKey = "identify-" + storeId + "-" + memberCode + "-" + System.currentTimeMillis();
        String url = integration.getSpiConfig().getIdentifyUrl();

        SpiRequest request = SpiRequest.builder()
            .spiType("MEMBER_IDENTIFY")
            .storeId(storeId)
            .idempotencyKey(idempotencyKey)
            .data(Map.of("member_code", memberCode))
            .build();

        return invokeSpi(url, request, integration.getSpiConfig().getSignSecret(),
            MemberIdentifyResult.class, "MEMBER_IDENTIFY", storeId);
    }

    /**
     * 权益扣减SPI调用（超时300ms，超时拒绝权益支付）
     */
    public DeductResult deductBenefits(String storeId, String orderId,
                                        String memberId, List<String> couponIds,
                                        int pointsToDeduct) {
        MemberIntegration integration = integrationRepository.findByStoreId(storeId)
            .orElseThrow(() -> new NotFoundException("未配置会员集成"));

        String idempotencyKey = "deduct-" + orderId;
        // 幂等检查：同一订单不重复扣减
        return idempotencyService.executeOnce(idempotencyKey, () -> {
            String url = integration.getSpiConfig().getDeductUrl();
            SpiRequest request = SpiRequest.builder()
                .spiType("DEDUCT")
                .storeId(storeId)
                .idempotencyKey(idempotencyKey)
                .data(Map.of(
                    "order_id", orderId,
                    "member_id", memberId,
                    "coupon_ids", couponIds,
                    "points_to_deduct", pointsToDeduct
                ))
                .build();
            return invokeSpi(url, request, integration.getSpiConfig().getSignSecret(),
                DeductResult.class, "DEDUCT", storeId);
        });
    }

    /**
     * 积分累积SPI调用（超时300ms，超时进入异步补偿）
     */
    @Async
    public void accumulatePoints(String storeId, String orderId,
                                  String memberId, int pointsToAdd) {
        MemberIntegration integration = integrationRepository.findByStoreId(storeId)
            .orElse(null);
        if (integration == null) return;

        String idempotencyKey = "accumulate-" + orderId;
        String url = integration.getSpiConfig().getAccumulateUrl();
        SpiRequest request = SpiRequest.builder()
            .spiType("ACCUMULATE")
            .storeId(storeId)
            .idempotencyKey(idempotencyKey)
            .data(Map.of("order_id", orderId, "member_id", memberId,
                "points_to_add", pointsToAdd))
            .build();

        try {
            invokeSpi(url, request, integration.getSpiConfig().getSignSecret(),
                Void.class, "ACCUMULATE", storeId);
        } catch (SpiTimeoutException e) {
            // 超时进入补偿队列，不影响主流程
            compensationTaskService.createTask("SPI_ACCUMULATE", idempotencyKey,
                request.toString());
        }
    }

    /**
     * 通用SPI调用（带签名、超时、日志）
     */
    private <T> T invokeSpi(String url, SpiRequest request, String signSecret,
                             Class<T> responseType, String spiType, String storeId) {
        long startMs = System.currentTimeMillis();
        SpiInvokeLog log = new SpiInvokeLog();
        log.setLogId(UUID.randomUUID().toString());
        log.setStoreId(storeId);
        log.setSpiType(spiType);
        log.setIdempotencyKey(request.getIdempotencyKey());
        log.setRequestBody(JsonUtils.toJson(request));
        log.setCreatedAt(LocalDateTime.now());

        try {
            // 添加签名
            String sign = HmacUtils.sign(JsonUtils.toJson(request.getData()), signSecret);
            request.setSign(sign);

            // 调用（300ms超时）
            T result = httpClient.postWithTimeout(url, request, responseType, 300);

            log.setStatus("SUCCESS");
            log.setResponseBody(JsonUtils.toJson(result));
            return result;
        } catch (TimeoutException e) {
            log.setStatus("TIMEOUT");
            throw new SpiTimeoutException("SPI调用超时: " + spiType);
        } catch (Exception e) {
            log.setStatus("FAILED");
            throw new SpiInvokeException("SPI调用失败: " + e.getMessage());
        } finally {
            log.setLatencyMs((int)(System.currentTimeMillis() - startMs));
            logRepository.save(log);
        }
    }
}
```

```java
        record.exceptionSource = command.getExceptionSource();
        record.businessType = command.getBusinessType();
        record.businessId = command.getBusinessId();
        record.businessNo = command.getBusinessNo();
        record.storeId = command.getStoreId();
        record.tenantId = command.getTenantId();
        record.exceptionTitle = command.getExceptionTitle();
        record.exceptionDescription = command.getExceptionDescription();
        record.exceptionData = command.getExceptionData();
        record.errorCode = command.getErrorCode();
        record.errorMessage = command.getErrorMessage();
        record.stackTrace = command.getStackTrace();
        record.handlingStatus = HandlingStatus.DETECTED;
        record.handlingPriority = calculatePriority(command.getSeverity());
        record.occurredAt = command.getOccurredAt();
        record.detectedAt = LocalDateTime.now();
        record.slaDeadline = calculateSlaDeadline(command.getSeverity());
        record.compensationRequired = false;

        record.addDomainEvent(new ExceptionDetectedEvent(record));

        return record;
    }

    /**
     * 分配处理人
     */
    public void assign(String assignee) {
        if (this.handlingStatus != HandlingStatus.DETECTED) {
            throw new IllegalStateException("异常状态不正确");
        }

        this.handlingStatus = HandlingStatus.ASSIGNED;
        this.assignedTo = assignee;
        this.assignedAt = LocalDateTime.now();

        this.addDomainEvent(new ExceptionAssignedEvent(this));
    }

    /**
     * 开始处理
     */
    public void startHandling() {
        if (this.handlingStatus != HandlingStatus.ASSIGNED) {
            throw new IllegalStateException("异常状态不正确");
        }

        this.handlingStatus = HandlingStatus.IN_PROGRESS;
    }

    /**
     * 添加处理备注
     */
    public void addHandlingNote(String noteBy, String noteContent) {
        if (this.handlingNotes == null) {
            this.handlingNotes = new ArrayList<>();
        }

        HandlingNote note = HandlingNote.builder()
            .noteTime(LocalDateTime.now())
            .noteBy(noteBy)
            .noteContent(noteContent)
            .build();

        this.handlingNotes.add(note);
    }

    /**
     * 解决异常
     */
    public void resolve(String resolutionAction, String resolutionResult, String resolvedBy) {
        if (this.handlingStatus != HandlingStatus.IN_PROGRESS) {
            throw new IllegalStateException("异常状态不正确");
        }

        this.handlingStatus = HandlingStatus.RESOLVED;
        this.resolutionAction = resolutionAction;
        this.resolutionResult = resolutionResult;
        this.resolvedAt = LocalDateTime.now();
        this.resolvedBy = resolvedBy;

        this.addDomainEvent(new ExceptionResolvedEvent(this));
    }

    /**
     * 标记需要补偿
     */
    public void markCompensationRequired(String compensationTaskId) {
        this.compensationRequired = true;
        this.compensationTaskId = compensationTaskId;
        this.compensationStatus = CompensationStatus.PENDING;
    }

    private static HandlingPriority calculatePriority(Severity severity) {
        switch (severity) {
            case CRITICAL:
                return HandlingPriority.URGENT;
            case HIGH:
                return HandlingPriority.HIGH;
            case MEDIUM:
                return HandlingPriority.NORMAL;
            default:
                return HandlingPriority.LOW;
        }
    }

    private static LocalDateTime calculateSlaDeadline(Severity severity) {
        LocalDateTime now = LocalDateTime.now();
        switch (severity) {
            case CRITICAL:
                return now.plusHours(1);
            case HIGH:
                return now.plusHours(4);
            case MEDIUM:
                return now.plusHours(24);
            default:
                return now.plusHours(72);
        }
    }
```

```java
        this.approvedAt = LocalDateTime.now();
        this.approvedBy = approver;
        this.approvalComment = comment;

        this.addDomainEvent(new CompensationTaskApprovedEvent(this));

     * 执行补偿
     */
    public void execute() {
        if (this.approvalStatus != ApprovalStatus.APPROVED) {
            throw new IllegalStateException("任务未审批通过");
        }

        if (this.taskStatus == TaskStatus.EXECUTING) {
            throw new IllegalStateException("任务正在执行中");
        }

        this.taskStatus = TaskStatus.EXECUTING;
        this.executionCount++;
        this.lastExecutionAt = LocalDateTime.now();

            this.taskStatus = TaskStatus.MANUAL_INTERVENTION;
            this.addDomainEvent(new CompensationFailedEvent(this));
        } else {
            this.taskStatus = TaskStatus.RETRYING;
        }
    }

        // 金额超过1000元需要审批
        if (command.getCompensationAmount() != null && command.getCompensationAmount() > 100000) {
            return true;
        }

        // 某些敏感操作需要审批
        if (command.getCompensationAction() == CompensationAction.ADJUST_ORDER_AMOUNT ||
            command.getCompensationAction() == CompensationAction.ADJUST_SETTLEMENT_AMOUNT) {
            return true;
        }

        return false;
    }
```

```java
@Service
@Transactional
public class ExceptionHandlingService {

    @Autowired
    private ExceptionRecordRepository exceptionRecordRepository;

    @Autowired
    private CompensationTaskRepository compensationTaskRepository;

    @Autowired
    private DomainEventPublisher eventPublisher;

    /**
     * 创建异常记录
     */
    public CreateExceptionRecordResponse createExceptionRecord(
        CreateExceptionRecordCommand command
    ) {
        ExceptionRecord record = ExceptionRecord.create(command);

        exceptionRecordRepository.save(record);
        eventPublisher.publish(record.getDomainEvents());

        // 严重异常立即分配
        if (record.getSeverity() == Severity.CRITICAL || record.getSeverity() == Severity.HIGH) {
            String assignee = findAvailableHandler(record.getExceptionType());
            record.assign(assignee);
            exceptionRecordRepository.save(record);

            // 发送告警通知
            sendAlertNotification(record);
        }

        return CreateExceptionRecordResponse.builder()
            .exceptionRecordId(record.getExceptionRecordId())
            .build();
    }

    /**
     * 解决异常
     */
    public void resolveException(
        String exceptionRecordId,
        String resolutionAction,
        String resolutionResult,
        String resolvedBy
    ) {
        ExceptionRecord record = exceptionRecordRepository.findById(exceptionRecordId)
            .orElseThrow(() -> new ExceptionRecordNotFoundException(exceptionRecordId));

        record.resolve(resolutionAction, resolutionResult, resolvedBy);

        exceptionRecordRepository.save(record);
        eventPublisher.publish(record.getDomainEvents());
    }

    /**
     * 创建补偿任务
     */
    public CreateCompensationTaskResponse createCompensationTask(
        CreateCompensationTaskCommand command
    ) {
        CompensationTask task = CompensationTask.create(command);

        compensationTaskRepository.save(task);
        eventPublisher.publish(task.getDomainEvents());

        // 关联到异常记录
        if (command.getExceptionRecordId() != null) {
            ExceptionRecord record = exceptionRecordRepository
                .findById(command.getExceptionRecordId())
                .orElseThrow();

            record.markCompensationRequired(task.getCompensationTaskId());
            exceptionRecordRepository.save(record);
        }

        return CreateCompensationTaskResponse.builder()
            .compensationTaskId(task.getCompensationTaskId())
            .build();
    }

    /**
     * 执行补偿任务
     */
    public void executeCompensationTask(String compensationTaskId) {
        CompensationTask task = compensationTaskRepository.findById(compensationTaskId)
            .orElseThrow(() -> new CompensationTaskNotFoundException(compensationTaskId));

        task.execute();
        compensationTaskRepository.save(task);

        try {
            // 执行补偿动作
            String result = performCompensationAction(task);

            task.markSuccess(result);
            compensationTaskRepository.save(task);

            eventPublisher.publish(task.getDomainEvents());

        } catch (Exception e) {
            log.error("补偿任务执行失败: {}", compensationTaskId, e);

            task.markFailed(e.getMessage());
            compensationTaskRepository.save(task);

            // 如果需要重试，加入重试队列
            if (task.getTaskStatus() == TaskStatus.RETRYING) {
                scheduleRetry(task);
            }
        }
    }

    private String performCompensationAction(CompensationTask task) {
        switch (task.getCompensationAction()) {
            case RETRY_ORDER_CREATION:
                return retryOrderCreation(task);
            case CANCEL_ORDER_WITH_REFUND:
                return cancelOrderWithRefund(task);
            case ADJUST_ORDER_AMOUNT:
                return adjustOrderAmount(task);
            case ISSUE_COUPON:
                return issueCoupon(task);
            default:
                throw new UnsupportedCompensationActionException(task.getCompensationAction());
        }
    }

    private String findAvailableHandler(ExceptionType exceptionType) {
        // 简单实现：返回默认处理人
        return "SYSTEM";
    }

    private void sendAlertNotification(ExceptionRecord record) {
        // 发送告警通知（邮件、短信、企业微信等）
        log.info("发送告警通知: exceptionRecordId={}", record.getExceptionRecordId());
    }

    private void scheduleRetry(CompensationTask task) {
        // 加入延迟队列，等待重试
        log.info("补偿任务加入重试队列: compensationTaskId={}", task.getCompensationTaskId());
    }
```

```java
@Service
public class AuditLogService {

    @Autowired
    private AuditLogRepository auditLogRepository;

    /**
     * 记录审计日志
     */
    public void recordAuditLog(CreateAuditLogCommand command) {
        AuditLog log = AuditLog.create(command);
        auditLogRepository.save(log);
    }

    /**
     * 查询审计日志
     */
    public PageResult<AuditLogDTO> queryAuditLogs(AuditLogQueryRequest request) {
        Specification<AuditLog> spec = buildSpecification(request);

        Pageable pageable = PageRequest.of(
            request.getPageNum() - 1,
            request.getPageSize(),
            Sort.by(Sort.Direction.DESC, "occurredAt")
        );

        Page<AuditLog> page = auditLogRepository.findAll(spec, pageable);

        List<AuditLogDTO> dtos = page.getContent().stream()
            .map(AuditLogDTO::from)
            .collect(Collectors.toList());

        return PageResult.of(dtos, page.getTotalElements());
    }

    private Specification<AuditLog> buildSpecification(AuditLogQueryRequest request) {
        return (root, query, cb) -> {
            List<Predicate> predicates = new ArrayList<>();

            if (request.getOperatorId() != null) {
                predicates.add(cb.equal(root.get("operatorId"), request.getOperatorId()));
            }

            if (request.getResourceType() != null) {
                predicates.add(cb.equal(root.get("resourceType"), request.getResourceType()));
            }

            if (request.getStartDate() != null) {
                predicates.add(cb.greaterThanOrEqualTo(root.get("occurredAt"), request.getStartDate()));
            }

            if (request.getEndDate() != null) {
                predicates.add(cb.lessThanOrEqualTo(root.get("occurredAt"), request.getEndDate()));
            }

            return cb.and(predicates.toArray(new Predicate[0]));
        };
    }
```

## 8. 本文结论

### 外部集成域服务

- **防腐层设计**：隔离外部系统差异，保护内部领域模型
- **适配器模式**：支持多配送平台扩展（达达/顺丰/闪送）
- **统一接口**：内部域只依赖统一接口（MemberService/DeliveryService/ERPService）
- **调用日志**：完整记录外部调用（请求/响应/耗时/状态）
- **SPI反向回调**：会员识别/权益扣减/积分累积，300ms超时降级策略
- **ERP集成**：支持多ERP系统（金蝶/用友），订单同步/库存同步
- **异常处理**：统一的异常处理机制

### 运营治理域服务

- **异常管理**：异常台账、SLA时限、自动分配、处理流程
- **补偿机制**：自动补偿、审批流程、重试策略、人工介入
- **审计日志**：完整记录、可追溯、按类型配置保留期限
- **监控告警**：告警规则配置、多渠道通知
- **接口设计**：REST API 和内部服务接口
- **数据库设计**：外部集成配置、调用日志、异常记录、补偿任务、审计日志完整表结构
