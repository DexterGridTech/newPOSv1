# 03-商品与门店经营域服务设计

## 1. 服务概述

商品与门店经营域服务（product-service）负责商品管理、门店管理、菜单管理等核心经营功能。

**核心职责**：
- 商品主数据管理
- 门店菜单管理
- 商品可售规则管理
- 渠道商品投影生成
- 商品快照服务
- 门店营业状态管理

**核心聚合根**：
- Product（商品）
- Menu（菜单）
- StoreOperatingConfig（门店营业配置）

### 3.2 Product 聚合根

```java
@Entity
@Table(name = "products")
public class Product {

    @Id
    private String productId;

    private String productCode;
    private String productName;

    @Enumerated(EnumType.STRING)
    private ProductType productType;

    private String categoryId;
    private String brandId;
    private String storeId;

    @Enumerated(EnumType.STRING)
    private OwnershipScope ownershipScope;

    private Long basePrice;

    @Enumerated(EnumType.STRING)
    private PriceUnit priceUnit;

    private String productImageUrl;

    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private List<String> productImages;

    private String description;

    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private List<ProductVariant> variants;

    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private ProductionProfile productionProfile;

    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private List<ModifierGroup> modifierGroups;

    @Enumerated(EnumType.STRING)
    private ProductStatus status;

    private Integer sortOrder;

    private LocalDateTime createdAt;
    private LocalDateTime updatedAt;

    @Version
    private Integer version;

    /**
     * 发布商品
     */
    public void publish() {
        if (this.status != ProductStatus.DRAFT) {
            throw new IllegalStateException("只有草稿状态的商品可以发布");
        }

        validateProduct();

        this.status = ProductStatus.ACTIVE;
        this.addDomainEvent(new ProductPublishedEvent(this));
    }

    /**
     * 下架商品
     */
    public void deactivate() {
        if (this.status != ProductStatus.ACTIVE) {
            throw new IllegalStateException("只有上架状态的商品可以下架");
        }

        this.status = ProductStatus.INACTIVE;
        this.addDomainEvent(new ProductDeactivatedEvent(this));
    }

    /**
     * 更新价格
     */
    public void updatePrice(Long newPrice) {
        if (newPrice <= 0) {
            throw new IllegalArgumentException("价格必须大于0");
        }

        Long oldPrice = this.basePrice;
        this.basePrice = newPrice;

        this.addDomainEvent(new ProductPriceChangedEvent(this, oldPrice, newPrice));
    }

    /**
     * 验证商品完整性
     */
    private void validateProduct() {
        if (StringUtils.isEmpty(productName)) {
            throw new IllegalStateException("商品名称不能为空");
        }
        if (basePrice == null || basePrice <= 0) {
            throw new IllegalStateException("商品价格必须大于0");
        }
        if (variants == null || variants.isEmpty()) {
            throw new IllegalStateException("商品必须至少有一个规格");
        }
    }

    /**
     * 生成商品快照
     */
    public ProductSnapshot createSnapshot(String variantId, List<String> modifierIds) {
        ProductVariant selectedVariant = findVariant(variantId);
        List<Modifier> selectedModifiers = findModifiers(modifierIds);

        Long finalPrice = calculateFinalPrice(selectedVariant, selectedModifiers);

        return ProductSnapshot.builder()
            .snapshotId(IdGenerator.generateSnapshotId())
            .productId(this.productId)
            .productCode(this.productCode)
            .productName(this.productName)
            .productType(this.productType)
            .selectedVariant(selectedVariant)
            .selectedModifiers(selectedModifiers)
            .unitPrice(finalPrice)
            .originalPrice(this.basePrice)
            .productImageUrl(this.productImageUrl)
            .productionProfile(this.productionProfile)
            .snapshotAt(LocalDateTime.now())
            .build();
    }

    private Long calculateFinalPrice(ProductVariant variant, List<Modifier> modifiers) {
        Long price = this.basePrice + variant.getPriceDelta();
        for (Modifier modifier : modifiers) {
            price += modifier.getPriceDelta();
        }
        return price;
    }
}
```

```java
@Entity
@Table(name = "menus")
public class Menu {

    @Id
    private String menuId;

    private String storeId;
    private String menuName;

    @Enumerated(EnumType.STRING)
    private MenuType menuType;

    private LocalDate effectiveDate;
    private LocalDate expireDate;

    @Enumerated(EnumType.STRING)
    private MenuStatus status;

    private Integer version;
    private String parentMenuId;

    @OneToMany(mappedBy = "menu", cascade = CascadeType.ALL)
    private List<MenuSection> sections;

    @Enumerated(EnumType.STRING)
    private ReviewStatus reviewStatus;

    private LocalDateTime publishedAt;
    private String publishedBy;

    /**
     * 提交审核
     */
    public void submitForReview() {
        if (this.status != MenuStatus.DRAFT) {
            throw new IllegalStateException("只有草稿状态的菜单可以提交审核");
        }

        validateMenu();

        this.reviewStatus = ReviewStatus.PENDING;
        this.addDomainEvent(new MenuSubmittedForReviewEvent(this));
    }

    /**
     * 审核通过
     */
    public void approve(String reviewerId) {
        if (this.reviewStatus != ReviewStatus.PENDING) {
            throw new IllegalStateException("只有待审核状态的菜单可以审核");
        }

        this.reviewStatus = ReviewStatus.APPROVED;
        this.addDomainEvent(new MenuApprovedEvent(this, reviewerId));
    }

    /**
     * 发布菜单
     */
    public void publish(String operatorId) {
        if (this.reviewStatus != ReviewStatus.APPROVED) {
            throw new IllegalStateException("只有审核通过的菜单可以发布");
        }

        this.status = MenuStatus.ACTIVE;
        this.publishedAt = LocalDateTime.now();
        this.publishedBy = operatorId;

        this.addDomainEvent(new MenuPublishedEvent(this));
    }

    /**
     * 停用菜单
     */
    public void deactivate() {
        this.status = MenuStatus.INACTIVE;
    }

    /**
     * 验证菜单完整性
     */
    private void validateMenu() {
        if (sections == null || sections.isEmpty()) {
            throw new IllegalStateException("菜单必须至少包含一个分区");
        }

        long totalProducts = sections.stream()
            .flatMap(section -> section.getProducts().stream())
            .count();

        if (totalProducts == 0) {
            throw new IllegalStateException("菜单必须至少包含一个商品");
        }
    }
}
```

```java
@Entity
@Table(name = "store_operating_configs")
public class StoreOperatingConfig {

    @Id
    private String configId;

    private String storeId;

    @Enumerated(EnumType.STRING)
    private OperatingStatus operatingStatus;

    private Boolean autoAcceptEnabled;
    private Integer acceptTimeoutSeconds;
    private Integer preparationBufferMinutes;
    private Integer maxConcurrentOrders;

    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private List<OperatingHours> operatingHours;

    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private List<SpecialOperatingDay> specialOperatingDays;

    private String pauseReason;
    private LocalDateTime pausedAt;
    private LocalDateTime resumeScheduledAt;

    /**
     * 开店
     */
    public void openStore() {
        if (this.operatingStatus == OperatingStatus.OPERATING) {
            throw new IllegalStateException("门店已经在营业中");
        }

        this.operatingStatus = OperatingStatus.OPERATING;
        this.pauseReason = null;
        this.pausedAt = null;

        this.addDomainEvent(new StoreOpenedEvent(this.storeId));
    }

    /**
     * 暂停营业
     */
    public void pauseStore(String reason) {
        if (this.operatingStatus != OperatingStatus.OPERATING) {
            throw new IllegalStateException("只有营业中的门店可以暂停");
        }

        this.operatingStatus = OperatingStatus.PAUSED;
        this.pauseReason = reason;
        this.pausedAt = LocalDateTime.now();

        this.addDomainEvent(new StorePausedEvent(this.storeId, reason));
    }

    /**
     * 恢复营业
     */
    public void resumeStore() {
        if (this.operatingStatus != OperatingStatus.PAUSED) {
            throw new IllegalStateException("只有暂停状态的门店可以恢复营业");
        }

        this.operatingStatus = OperatingStatus.OPERATING;
        this.pauseReason = null;
        this.pausedAt = null;
        this.resumeScheduledAt = null;

        this.addDomainEvent(new StoreResumedEvent(this.storeId));
    }

    /**
     * 检查当前是否在营业时间
     */
    public boolean isOperatingNow() {
        if (this.operatingStatus != OperatingStatus.OPERATING) {
            return false;
        }

        LocalDateTime now = LocalDateTime.now();
        DayOfWeek dayOfWeek = now.getDayOfWeek();
        LocalTime currentTime = now.toLocalTime();

        return operatingHours.stream()
            .filter(hours -> hours.getDayOfWeek() == dayOfWeek.getValue())
            .flatMap(hours -> hours.getTimeSlots().stream())
            .anyMatch(slot -> isTimeInSlot(currentTime, slot));
    }

    private boolean isTimeInSlot(LocalTime time, TimeSlot slot) {
        LocalTime start = LocalTime.parse(slot.getStartTime());
        LocalTime end = LocalTime.parse(slot.getEndTime());
        return !time.isBefore(start) && !time.isAfter(end);
    }
}
```

### 5.1 ProductApplicationService

```java
@Service
@Transactional
public class ProductApplicationService {

    @Autowired
    private ProductRepository productRepository;

    @Autowired
    private ProductAvailabilityService productAvailabilityService;

    @Autowired
    private DomainEventPublisher eventPublisher;

    /**
     * 创建商品
     */
    public CreateProductResponse createProduct(CreateProductRequest request) {
        // 验证请求
        validateCreateProductRequest(request);

        // 创建商品聚合
        Product product = Product.builder()
            .productId(IdGenerator.generateProductId())
            .productCode(request.getProductCode())
            .productName(request.getProductName())
            .productType(request.getProductType())
            .categoryId(request.getCategoryId())
            .brandId(request.getBrandId())
            .storeId(request.getStoreId())
            .ownershipScope(request.getOwnershipScope())
            .basePrice(request.getBasePrice())
            .priceUnit(request.getPriceUnit())
            .productImageUrl(request.getProductImageUrl())
            .productImages(request.getProductImages())
            .description(request.getDescription())
            .variants(request.getVariants())
            .productionProfile(request.getProductionProfile())
            .modifierGroups(request.getModifierGroups())
            .status(ProductStatus.DRAFT)
            .sortOrder(request.getSortOrder())
            .createdAt(LocalDateTime.now())
            .build();

        // 保存商品
        productRepository.save(product);

        // 发布领域事件
        eventPublisher.publish(product.getDomainEvents());

        return CreateProductResponse.builder()
            .productId(product.getProductId())
            .productCode(product.getProductCode())
            .build();
    }

    /**
     * 发布商品
     */
    public void publishProduct(String productId) {
        Product product = productRepository.findById(productId)
            .orElseThrow(() -> new ProductNotFoundException(productId));

        product.publish();

        productRepository.save(product);
        eventPublisher.publish(product.getDomainEvents());
    }

    /**
     * 更新商品价格
     */
```

### 6.1 ProductAvailabilityService（可售规则引擎 AvailabilityRuleEngine）

```java
@Service
public class ProductAvailabilityService {

    @Autowired
    private AvailabilityRuleRepository ruleRepository;

    @Autowired
    private StoreOperatingConfigRepository configRepository;

    /**
     * 检查商品可售性
     */
    public boolean checkAvailability(
        String productId,
        String storeId,
        ChannelType channelType,
        LocalDateTime checkTime
    ) {
        // 1. 检查门店营业状态
        StoreOperatingConfig config = configRepository.findByStoreId(storeId)
            .orElseThrow(() -> new StoreNotFoundException(storeId));

        if (!config.isOperatingNow()) {
            return false;
        }

        // 2. 查询可售规则
        List<AvailabilityRule> rules = ruleRepository.findByProductIdAndStoreId(
            productId, storeId
        );

        // 3. 评估所有规则
        for (AvailabilityRule rule : rules) {
            if (!rule.isActive()) {
                continue;
            }

            if (!evaluateRule(rule, channelType, checkTime)) {
                return false;
            }
        }

        return true;
    }

    private boolean evaluateRule(
        AvailabilityRule rule,
        ChannelType channelType,
        LocalDateTime checkTime
    ) {
        switch (rule.getRuleType()) {
            case TIME_SLOT:
                return evaluateTimeSlotRule(rule, checkTime);
            case CHANNEL:
                return evaluateChannelRule(rule, channelType);
            case STOCK:
                return evaluateStockRule(rule);
            case QUOTA:
                return evaluateQuotaRule(rule);
            default:
                return true;
        }
    }

    private boolean evaluateTimeSlotRule(AvailabilityRule rule, LocalDateTime checkTime) {
        TimeSlotRuleConfig config = rule.getTimeSlotConfig();
        LocalTime time = checkTime.toLocalTime();
        DayOfWeek dayOfWeek = checkTime.getDayOfWeek();

        return config.getAllowedTimeSlots().stream()
            .anyMatch(slot ->
                slot.getDaysOfWeek().contains(dayOfWeek.getValue()) &&
                isTimeInRange(time, slot.getStartTime(), slot.getEndTime())
            );
    }

    private boolean evaluateChannelRule(AvailabilityRule rule, ChannelType channelType) {
        ChannelRuleConfig config = rule.getChannelConfig();

        if (config.getAllowedChannels() != null && !config.getAllowedChannels().isEmpty()) {
            return config.getAllowedChannels().contains(channelType);
        }

        if (config.getBlockedChannels() != null && !config.getBlockedChannels().isEmpty()) {
            return !config.getBlockedChannels().contains(channelType);
        }

        return true;
    }

    private boolean evaluateStockRule(AvailabilityRule rule) {
        StockRuleConfig config = rule.getStockConfig();

        if (config.getStockType() == StockType.SOLD_OUT) {
            return false;
        }

        if (config.getStockType() == StockType.LIMITED) {
            return config.getCurrentStock() > 0;
        }

        return true;
    }

    private boolean evaluateQuotaRule(AvailabilityRule rule) {
        QuotaRuleConfig config = rule.getQuotaConfig();
        return config.getQuotaUsed() < config.getQuotaLimit();
    }

    private boolean isTimeInRange(LocalTime time, String startStr, String endStr) {
        LocalTime start = LocalTime.parse(startStr);
        LocalTime end = LocalTime.parse(endStr);
        return !time.isBefore(start) && !time.isAfter(end);
    }
}
```

### 6.2 ChannelProductSyncService（渠道商品同步服务）

```java
@Service
public class ChannelProductSyncService {

    @Autowired
    private ChannelProductMappingRepository mappingRepository;

    @Autowired
    private ChannelShopBindingRepository bindingRepository;

    @Autowired
    private ProductRepository productRepository;

    /**
     * 同步商品到渠道
     */
    public SyncResult syncProductToChannel(
        String productId,
        String storeId,
        ChannelType channelType
    ) {
        // 1. 查询商品
        Product product = productRepository.findById(productId)
            .orElseThrow(() -> new ProductNotFoundException(productId));

        // 2. 查询渠道绑定
        ChannelShopBinding binding = bindingRepository.findByStoreIdAndChannelType(
            storeId, channelType
        ).orElseThrow(() -> new ChannelBindingNotFoundException(storeId, channelType));

        // 3. 查询或创建商品映射
        ChannelProductMapping mapping = mappingRepository
            .findByProductIdAndChannelType(productId, channelType)
            .orElseGet(() -> createNewMapping(product, binding));

        // 4. 生成渠道商品投影
        ChannelProductProjection projection = generateProjection(product, mapping);

        // 5. 调用渠道API同步
        try {
            ChannelAdapter adapter = getChannelAdapter(channelType);
            String externalProductId = adapter.syncProduct(binding, projection);

            // 6. 更新映射状态
            mapping.setSyncStatus(SyncStatus.SYNCED);
            mapping.setExternalProductId(externalProductId);
            mapping.setLastSyncAt(LocalDateTime.now());
            mappingRepository.save(mapping);

            return SyncResult.success(externalProductId);

        } catch (Exception e) {
            // 记录同步失败
            mapping.setSyncStatus(SyncStatus.SYNC_FAILED);
            mapping.setSyncErrorMessage(e.getMessage());
            mappingRepository.save(mapping);

            return SyncResult.failure(e.getMessage());
        }
    }

    private ChannelProductProjection generateProjection(
        Product product,
        ChannelProductMapping mapping
    ) {
        FieldMappingConfig config = mapping.getFieldMappingConfig();

        return ChannelProductProjection.builder()
            .productName(applyNameMapping(product.getProductName(), config.getNameMapping()))
            .price(applyPriceMapping(product.getBasePrice(), config.getPriceMapping()))
            .imageUrl(applyImageMapping(product.getProductImageUrl(), config.getImageMapping()))
            .variants(product.getVariants())
            .modifierGroups(applyModifierMapping(product.getModifierGroups(), config.getModifierMapping()))
            .build();
    }

    private ChannelProductMapping createNewMapping(Product product, ChannelShopBinding binding) {
        ChannelProductMapping mapping = new ChannelProductMapping();
        mapping.setMappingId(IdGenerator.generateMappingId());
        mapping.setProductId(product.getProductId());
        mapping.setChannelType(binding.getChannelType());
        mapping.setSyncStatus(SyncStatus.NOT_SYNCED);
        mapping.setFieldMappingConfig(FieldMappingConfig.defaultConfig());
        mapping.setCreatedAt(LocalDateTime.now());
        return mappingRepository.save(mapping);
    }

    private String applyNameMapping(String productName, NameMapping nameMapping) {
        if (nameMapping == null || Boolean.TRUE.equals(nameMapping.getUseOriginal())) {
            return productName;
        }
        String result = productName;
        if (nameMapping.getPrefix() != null) {
            result = nameMapping.getPrefix() + result;
        }
        if (nameMapping.getSuffix() != null) {
            result = result + nameMapping.getSuffix();
        }
        return result;
    }

    private Long applyPriceMapping(Long basePrice, PriceMapping priceMapping) {
        if (priceMapping == null || "ORIGINAL".equals(priceMapping.getStrategy())) {
            return basePrice;
        }
        if ("MARKUP".equals(priceMapping.getStrategy()) && priceMapping.getPriceDelta() != null) {
            return basePrice + priceMapping.getPriceDelta();
        }
        if ("MARKDOWN".equals(priceMapping.getStrategy()) && priceMapping.getPriceDelta() != null) {
            return basePrice - priceMapping.getPriceDelta();
        }
        return basePrice;
    }

    private String applyImageMapping(String imageUrl, ImageMapping imageMapping) {
        if (imageMapping == null || "ORIGINAL".equals(imageMapping.getStrategy())) {
            return imageUrl;
        }
        if ("REPLACE".equals(imageMapping.getStrategy()) && imageMapping.getOverrideUrl() != null) {
            return imageMapping.getOverrideUrl();
        }
        return imageUrl;
    }

    private List<ModifierGroup> applyModifierMapping(
        List<ModifierGroup> groups, ModifierMapping modifierMapping
    ) {
        if (modifierMapping == null || !Boolean.TRUE.equals(modifierMapping.getSyncModifiers())) {
            return Collections.emptyList();
        }
        if (modifierMapping.getExcludedGroupIds() != null) {
            return groups.stream()
                .filter(g -> !modifierMapping.getExcludedGroupIds().contains(g.getGroupId()))
                .collect(Collectors.toList());
        }
        return groups;
    }

    private ChannelAdapter getChannelAdapter(ChannelType channelType) {
        // 通过Spring容器获取对应渠道适配器
        return channelAdapters.get(channelType);
    }
}
```

### 9.2 菜单查询优化

```java
@Service
public class MenuQueryOptimizationService {

    @Autowired
    private MenuRepository menuRepository;

    @Autowired
    private RedisTemplate<String, String> redisTemplate;

    /**
     * 查询门店当前菜单（带缓存）
     */
    public MenuDTO getCurrentMenu(String storeId) {
        String cacheKey = "menu:current:" + storeId;

        String cachedJson = redisTemplate.opsForValue().get(cacheKey);
        if (cachedJson != null) {
            return JSON.parseObject(cachedJson, MenuDTO.class);
        }

        Menu menu = menuRepository.findCurrentActiveMenu(storeId)
            .orElseThrow(() -> new MenuNotFoundException(storeId));

        MenuDTO dto = MenuDTO.from(menu);

        // 缓存1小时
        redisTemplate.opsForValue().set(
            cacheKey,
            JSON.toJSONString(dto),
            1,
            TimeUnit.HOURS
        );

        return dto;
    }

    /**
     * 清除菜单缓存
     */
    public void evictMenuCache(String storeId) {
        String cacheKey = "menu:current:" + storeId;
        redisTemplate.delete(cacheKey);
    }
}
```

- **领域模型**：Product、Menu、StoreOperatingConfig 聚合根设计，含完整状态机和业务规则

- **应用服务**：商品创建/发布/下架、菜单创建/审核/发布、门店开店/暂停/恢复等核心操作

- **事件处理**：商品/菜单/门店领域事件，集成 RocketMQ 消息总线

- **性能优化**：商品 Caffeine 缓存 + 菜单 Redis 缓存双层缓存架构
- **数据库**：7 张核心表，完整 DDL 含索引和约束
