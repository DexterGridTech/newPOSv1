# 05-渠道集成域服务设计

## 1. 服务概述

渠道集成域服务（channel-service）负责与外部渠道平台的集成，包括订单接收、状态回传、商品同步等功能。

**核心职责**：
- 渠道店铺绑定管理
- 外部订单接收与翻译
- 订单状态回传
- 商品同步到渠道
- Token 管理与刷新
- 渠道操作日志记录

**核心聚合根**：
- ChannelShopBinding（渠道店铺绑定）
- ChannelOrder（渠道订单）
- ChannelProductMapping（渠道商品映射）

```java
@Entity
@Table(name = "channel_shop_bindings")
public class ChannelShopBinding {

    @Id
    private String bindingId;

    private String storeId;

    @Enumerated(EnumType.STRING)
    private ChannelType channelType;

    private String externalShopId;
    private String externalShopName;
    private String externalPoiCode;

    // ISV授权信息（加密存储）
    private String appKey;
    private String accessToken;
    private String refreshToken;
    private String tokenType;
    private LocalDateTime tokenExpiresAt;

    @Enumerated(EnumType.STRING)
    private BindingStatus bindingStatus;

    @Enumerated(EnumType.STRING)
    private AuthorizationStatus authorizationStatus;

    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private SyncConfig syncConfig;

    private LocalDateTime boundAt;
    private LocalDateTime authorizedAt;
    private LocalDateTime lastTokenRefreshAt;

    @Version
    private Integer version;

    /**
     * 授权绑定
     */
    public void authorize(String accessToken, String refreshToken, LocalDateTime expiresAt) {
        if (this.bindingStatus != BindingStatus.PENDING_AUTHORIZATION) {
            throw new IllegalStateException("绑定状态不正确");
        }

        this.accessToken = encryptToken(accessToken);
        this.refreshToken = encryptToken(refreshToken);
        this.tokenExpiresAt = expiresAt;
        this.authorizationStatus = AuthorizationStatus.AUTHORIZED;
        this.authorizedAt = LocalDateTime.now();

        this.addDomainEvent(new ChannelBindingAuthorizedEvent(this));
    }

    /**
     * 激活绑定
     */
    public void activate() {
        if (this.authorizationStatus != AuthorizationStatus.AUTHORIZED) {
            throw new IllegalStateException("必须先完成授权");
        }

        this.bindingStatus = BindingStatus.ACTIVE;
        this.addDomainEvent(new ChannelBindingActivatedEvent(this));
    }

    /**
     * 刷新Token
     */
    public void refreshToken(String newAccessToken, String newRefreshToken, LocalDateTime newExpiresAt) {
        this.accessToken = encryptToken(newAccessToken);
        this.refreshToken = encryptToken(newRefreshToken);
        this.tokenExpiresAt = newExpiresAt;
        this.lastTokenRefreshAt = LocalDateTime.now();

        this.addDomainEvent(new ChannelTokenRefreshedEvent(this));
    }

    /**
     * Token过期
     */
    public void markTokenExpired() {
        this.authorizationStatus = AuthorizationStatus.EXPIRED;
        this.addDomainEvent(new ChannelTokenExpiredEvent(this));
    }

    /**
     * 暂停绑定
     */
    public void suspend() {
        this.bindingStatus = BindingStatus.SUSPENDED;
        this.addDomainEvent(new ChannelBindingSuspendedEvent(this));
    }

    /**
     * 解除绑定
     */
    public void unbind() {
        this.bindingStatus = BindingStatus.UNBOUND;
        this.authorizationStatus = AuthorizationStatus.REVOKED;
        this.addDomainEvent(new ChannelBindingUnboundEvent(this));
    }

    /**
     * 检查Token是否即将过期（7天内）
     */
    public boolean isTokenExpiringSoon() {
        if (this.tokenExpiresAt == null) {
            return false;
        }
        return LocalDateTime.now().plusDays(7).isAfter(this.tokenExpiresAt);
    }

    /**
     * 检查Token是否已过期
     */
    public boolean isTokenExpired() {
        if (this.tokenExpiresAt == null) {
            return true;
        }
        return LocalDateTime.now().isAfter(this.tokenExpiresAt);
    }

    private String encryptToken(String token) {
        // 使用AES加密
        return AESUtil.encrypt(token);
    }

    public String getDecryptedAccessToken() {
        return AESUtil.decrypt(this.accessToken);
    }

    public String getDecryptedRefreshToken() {
        return AESUtil.decrypt(this.refreshToken);
    }
}
```

```java
@Entity
@Table(name = "channel_orders")
public class ChannelOrder {

    @Id
    private String channelOrderId;

    private String bindingId;
    private String storeId;

    @Enumerated(EnumType.STRING)
    private ChannelType channelType;

    private String externalOrderNo;
    private String externalOrderStatus;

    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private JsonNode rawOrderData;

    private LocalDateTime receivedAt;
    private LocalDateTime parsedAt;

    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private UnifiedOrderPayload unifiedOrderPayload;

    @Enumerated(EnumType.STRING)
    private ProcessingStatus processingStatus;

    private String internalOrderId;
    private String errorCode;
    private String errorMessage;
    private Integer retryCount;

    private LocalDateTime channelAcceptDeadline;
    private LocalDateTime channelDeliveryTime;

    private String idempotencyKey;

    @Version
    private Integer version;

    /**
     * 创建渠道订单
     */
    public static ChannelOrder create(
        ChannelShopBinding binding,
        String externalOrderNo,
        JsonNode rawData
    ) {
        ChannelOrder order = new ChannelOrder();
        order.channelOrderId = IdGenerator.generateChannelOrderId();
        order.bindingId = binding.getBindingId();
        order.storeId = binding.getStoreId();
        order.channelType = binding.getChannelType();
        order.externalOrderNo = externalOrderNo;
        order.rawOrderData = rawData;
        order.processingStatus = ProcessingStatus.RECEIVED;
        order.receivedAt = LocalDateTime.now();
        order.idempotencyKey = generateIdempotencyKey(binding.getChannelType(), externalOrderNo);
        order.retryCount = 0;

        order.addDomainEvent(new ChannelOrderReceivedEvent(order));

        return order;
    }

    /**
     * 解析订单
     */
    public void parse(UnifiedOrderPayload payload) {
        if (this.processingStatus != ProcessingStatus.RECEIVED) {
            throw new IllegalStateException("订单状态不正确");
        }

        this.unifiedOrderPayload = payload;
        this.processingStatus = ProcessingStatus.PARSED;
        this.parsedAt = LocalDateTime.now();

        this.addDomainEvent(new ChannelOrderParsedEvent(this));
    }

    /**
     * 订单创建成功
     */
    public void markOrderCreated(String internalOrderId) {
        if (this.processingStatus != ProcessingStatus.PARSED) {
            throw new IllegalStateException("订单状态不正确");
        }

        this.internalOrderId = internalOrderId;
        this.processingStatus = ProcessingStatus.ORDER_CREATED;

        this.addDomainEvent(new InternalOrderCreatedEvent(this));
    }

    /**
     * 处理失败
     */
    public void markFailed(String errorCode, String errorMessage) {
        this.processingStatus = ProcessingStatus.CREATE_FAILED;
        this.errorCode = errorCode;
        this.errorMessage = errorMessage;
        this.retryCount++;

        this.addDomainEvent(new ChannelOrderProcessFailedEvent(this));
    }

    /**
     * 标记为待重试
     */
    public void markRetryPending() {
        this.processingStatus = ProcessingStatus.RETRY_PENDING;
    }

    private static String generateIdempotencyKey(ChannelType channelType, String externalOrderNo) {
        return channelType.name() + ":" + externalOrderNo;
    }
}
```

### 5.2 MeituanAdapter（美团适配器完整实现）

```java
@Component
public class MeituanAdapter implements ChannelAdapter {

    private static final String MEITUAN_API_BASE = "https://waimaiopen.meituan.com/api/v1";

    @Autowired
    private RestTemplate restTemplate;

    @Override
    public ChannelType getChannelType() {
        return ChannelType.MEITUAN;
    }

    /**
     * 同步商品到美团
     */
    @Override
    public String syncProduct(ChannelShopBinding binding, ChannelProductProjection projection) {
        String url = MEITUAN_API_BASE + "/food/save";

        Map<String, Object> params = new HashMap<>();
        params.put("app_id", binding.getAppKey());
        params.put("epoiId", binding.getExternalShopId());
        params.put("name", projection.getProductName());
        params.put("price", convertFenToYuan(projection.getPrice()));
        params.put("picture", projection.getImageUrl());

        // 构建规格信息
        if (projection.getVariants() != null && !projection.getVariants().isEmpty()) {
            List<Map<String, Object>> skuList = projection.getVariants().stream()
                .map(variant -> {
                    Map<String, Object> sku = new HashMap<>();
                    sku.put("skuId", variant.getVariantId());
                    sku.put("spec", variant.getVariantName());
                    sku.put("price", convertFenToYuan(
                        projection.getPrice() + variant.getPriceDelta()));
                    return sku;
                })
                .collect(Collectors.toList());
            params.put("skus", JSON.toJSONString(skuList));
        }

        // 签名
        String sign = MeituanSignatureUtil.sign(params, binding.getDecryptedAccessToken());
        params.put("sig", sign);

        // 调用 API
        ResponseEntity<String> response = restTemplate.postForEntity(url, params, String.class);
        JsonNode result = JSON.parseObject(response.getBody(), JsonNode.class);

        if ("ok".equals(result.get("data").asText())) {
            return result.has("food_id") ? result.get("food_id").asText() : "synced";
        } else {
            throw new ChannelApiException("美团商品同步失败: " + result.toString());
        }
    }

    /**
     * 回传订单状态到美团
     */
    @Override
    public ChannelStatusSyncResponse syncOrderStatus(ChannelStatusSyncRequest request) {
        String url;
        Map<String, Object> params = new HashMap<>();

        switch (request.getStatus()) {
            case ACCEPTED:
                url = MEITUAN_API_BASE + "/order/confirm";
                params.put("order_id", request.getExternalOrderNo());
                break;
            case PRODUCTION_COMPLETED:
                url = MEITUAN_API_BASE + "/order/preparing/completedByShop";
                params.put("order_id", request.getExternalOrderNo());
                break;
            case DELIVERING:
                url = MEITUAN_API_BASE + "/order/delivering";
                params.put("order_id", request.getExternalOrderNo());
                break;
            default:
                return ChannelStatusSyncResponse.builder()
                    .success(true)
                    .build();
        }

        // 签名
        String sign = MeituanSignatureUtil.sign(params, request.getAccessToken());
        params.put("sig", sign);

        ResponseEntity<String> response = restTemplate.postForEntity(url, params, String.class);
        JsonNode result = JSON.parseObject(response.getBody(), JsonNode.class);

        return ChannelStatusSyncResponse.builder()
            .success("ok".equals(result.get("data").asText()))
            .errorCode(result.has("error") ? result.get("error").get("code").asText() : null)
            .errorMessage(result.has("error") ? result.get("error").get("msg").asText() : null)
            .build();
    }

    /**
     * 刷新美团 Token
     */
    @Override
    public TokenRefreshResult refreshToken(String appKey, String refreshToken) {
        String url = MEITUAN_API_BASE + "/oauth/token/refresh";

        Map<String, Object> params = new HashMap<>();
        params.put("app_id", appKey);
        params.put("refresh_token", refreshToken);
        params.put("grant_type", "refresh_token");

        ResponseEntity<String> response = restTemplate.postForEntity(url, params, String.class);
        JsonNode result = JSON.parseObject(response.getBody(), JsonNode.class);

        if (result.has("access_token")) {
            return TokenRefreshResult.builder()
                .accessToken(result.get("access_token").asText())
                .refreshToken(result.get("refresh_token").asText())
                .expiresAt(LocalDateTime.now().plusSeconds(result.get("expires_in").asLong()))
                .success(true)
                .build();
        } else {
            return TokenRefreshResult.builder()
                .success(false)
                .errorMessage(result.toString())
                .build();
        }
    }

    private Double convertFenToYuan(Long fen) {
        return fen / 100.0;
    }
}

/**
 * 美团签名工具
 */
public class MeituanSignatureUtil {

    public static String sign(Map<String, Object> params, String accessToken) {
        // 1. 按 key 字母序排列
        TreeMap<String, Object> sorted = new TreeMap<>(params);

        // 2. 拼接参数
        StringBuilder sb = new StringBuilder();
        for (Map.Entry<String, Object> entry : sorted.entrySet()) {
            sb.append(entry.getKey()).append("=").append(entry.getValue()).append("&");
        }
        sb.append("access_token=").append(accessToken);

        // 3. MD5 签名
        return DigestUtils.md5DigestAsHex(sb.toString().getBytes(StandardCharsets.UTF_8));
    }
}

/**
 * 美团签名验证器
 */
public class MeituanSignatureVerifier {

    public static boolean verify(ReceiveChannelOrderRequest request) {
        // 验证美团推送的签名
        String expectedSign = MeituanSignatureUtil.sign(
            extractParamsFromRequest(request),
            getAccessTokenForStore(request.getStoreId())
        );
        return expectedSign.equals(request.getSignature());
    }
}
```

```java
@Component
public class ElemeAdapter implements ChannelAdapter {

    private static final String ELEME_API_BASE = "https://open-api.shop.ele.me/api/v1";

    @Autowired
    private RestTemplate restTemplate;

    @Override
    public ChannelType getChannelType() {
        return ChannelType.ELEME;

    /**
     * 同步商品到饿了么
     */
    @Override
    public String syncProduct(ChannelShopBinding binding, ChannelProductProjection projection) {
        String url = ELEME_API_BASE + "/";

        Map<String, Object> body = new HashMap<>();
        body.put("action", "eleme.product.item.createItem");
        body.put("token", binding.getDecryptedAccessToken());

        Map<String, Object> params = new HashMap<>();
        params.put("shopId", binding.getExternalShopId());
        params.put("name", projection.getProductName());
        params.put("price", convertFenToYuan(projection.getPrice()));
        params.put("imageHash", projection.getImageUrl());

        // 构建规格信息
        if (projection.getVariants() != null && !projection.getVariants().isEmpty()) {
            List<Map<String, Object>> specs = projection.getVariants().stream()
                .map(variant -> {
                    Map<String, Object> spec = new HashMap<>();
                    spec.put("name", variant.getVariantName());
                    spec.put("price", convertFenToYuan(
                        projection.getPrice() + variant.getPriceDelta()));
                    return spec;
                })
                .collect(Collectors.toList());
            params.put("specs", specs);
        }

        body.put("params", params);

        // 签名
        String sign = ElemeSignatureUtil.sign(body, binding.getAppKey());
        body.put("signature", sign);

        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);
        HttpEntity<Map<String, Object>> entity = new HttpEntity<>(body, headers);

        ResponseEntity<String> response = restTemplate.postForEntity(url, entity, String.class);
        JsonNode result = JSON.parseObject(response.getBody(), JsonNode.class);

        if (result.has("result")) {
            return result.get("result").get("itemId").asText();
        } else {
            throw new ChannelApiException("饿了么商品同步失败: " + result.toString());
        }

                body.put("action", "eleme.order.agreeRefund");
                // 饿了么没有直接的"制作完成"API，使用自定义状态
                body.put("action", "eleme.order.completedCooking");
                break;
            case DELIVERING:
                body.put("action", "eleme.order.delivering");
                break;
            default:
                return ChannelStatusSyncResponse.builder().success(true).build();
        }

        body.put("token", request.getAccessToken());

        Map<String, Object> params = new HashMap<>();
        params.put("orderId", request.getExternalOrderNo());
        body.put("params", params);

        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);
        HttpEntity<Map<String, Object>> entity = new HttpEntity<>(body, headers);

        ResponseEntity<String> response = restTemplate.postForEntity(url, entity, String.class);
        JsonNode result = JSON.parseObject(response.getBody(), JsonNode.class);

        return ChannelStatusSyncResponse.builder()
            .success(!result.has("error"))
            .errorCode(result.has("error") ? result.get("error").get("code").asText() : null)
            .errorMessage(result.has("error") ? result.get("error").get("message").asText() : null)
            .build();

        params.put("grant_type", "refresh_token");
        params.put("refresh_token", refreshToken);
        params.put("client_id", appKey);

        ResponseEntity<String> response = restTemplate.postForEntity(url, params, String.class);

                .refreshToken(result.get("refresh_token").asText())

                .build();
        } else {

/**
```

```java
 * 统一管理与各渠道平台 ISV 接口的交互

@Service
public class ISVProxyService {

    @Autowired
    private Map<ChannelType, ChannelAdapter> adapters;

    @Autowired
    private ChannelShopBindingRepository bindingRepository;

    @Autowired
    private ChannelOperationLogRepository logRepository;

    /**
     * 代理调用渠道API（带重试和日志）
     */
    public <T> T executeWithRetry(
        String bindingId,
        String operationType,
        Function<ChannelShopBinding, T> operation,
        int maxRetries
    ) {
        ChannelShopBinding binding = bindingRepository.findById(bindingId)
            .orElseThrow(() -> new ChannelBindingNotFoundException(bindingId));

        int retryCount = 0;
        Exception lastException = null;

        while (retryCount <= maxRetries) {
            long startTime = System.currentTimeMillis();
            try {
                T result = operation.apply(binding);
                long duration = System.currentTimeMillis() - startTime;

                // 记录成功日志
                recordLog(binding, operationType, null, result, duration, ExecutionStatus.SUCCESS);
                return result;

            } catch (TokenExpiredException e) {
                // Token 过期，尝试刷新
                refreshBindingToken(binding);
                binding = bindingRepository.findById(bindingId).orElseThrow();
                retryCount++;
                lastException = e;

            } catch (Exception e) {
                long duration = System.currentTimeMillis() - startTime;
                recordLog(binding, operationType, null, null, duration, ExecutionStatus.FAILED);
                retryCount++;
                lastException = e;

                if (retryCount <= maxRetries) {
                    try {
                        Thread.sleep(1000L * retryCount); // 指数退避
                    } catch (InterruptedException ie) {
                        Thread.currentThread().interrupt();
                    }
                }
            }
        }

        throw new ChannelApiException("渠道API调用失败（已重试" + maxRetries + "次）", lastException);
    }

    /**
     * 刷新绑定的 Token
     */
    private void refreshBindingToken(ChannelShopBinding binding) {
        ChannelAdapter adapter = adapters.get(binding.getChannelType());

        TokenRefreshResult result = adapter.refreshToken(
            binding.getAppKey(),
            binding.getDecryptedRefreshToken()
        );

        if (result.isSuccess()) {
            binding.refreshToken(
                result.getAccessToken(),
                result.getRefreshToken(),
                result.getExpiresAt()
            );
            bindingRepository.save(binding);
        } else {
            binding.markTokenExpired();
            bindingRepository.save(binding);
            throw new TokenExpiredException("Token刷新失败: " + result.getErrorMessage());
        }
    }

    private void recordLog(
        ChannelShopBinding binding,
        String operationType,
        Object request,
        Object response,
        long duration,
        ExecutionStatus status
    ) {
        ChannelOperationLog log = ChannelOperationLog.builder()
            .logId(IdGenerator.generateLogId())
            .bindingId(binding.getBindingId())
            .channelType(binding.getChannelType())
            .operationType(operationType)
            .requestPayload(request != null ? JSON.toJSONString(request) : null)
            .responsePayload(response != null ? JSON.toJSONString(response) : null)
            .durationMs(duration)
            .executionStatus(status)
            .createdAt(LocalDateTime.now())
            .build();

        logRepository.save(log);
    }
}
```

```java
@Service
@Transactional
public class ChannelOrderApplicationService {

    @Autowired
    private ChannelOrderRepository channelOrderRepository;

    @Autowired
    private ChannelShopBindingRepository bindingRepository;

    @Autowired
    private ChannelOrderParserService parserService;

    @Autowired
    private OrderApplicationService orderApplicationService;

    private DomainEventPublisher eventPublisher;

    /**
     * 接收渠道订单
     */
    public ReceiveChannelOrderResponse receiveChannelOrder(
        ReceiveChannelOrderRequest request
    ) {
        // 1. 验证签名
        if (!verifySignature(request)) {
            throw new InvalidSignatureException();
        }

        // 2. 幂等检查
        String idempotencyKey = generateIdempotencyKey(
            request.getChannelType(),
            request.getExternalOrderNo()
        );

        if (channelOrderRepository.existsByIdempotencyKey(idempotencyKey)) {
            log.info("订单已存在，跳过处理: {}", idempotencyKey);
            return ReceiveChannelOrderResponse.success();
        }

        // 3. 查询渠道绑定
        ChannelShopBinding binding = bindingRepository
            .findByStoreIdAndChannelType(request.getStoreId(), request.getChannelType())
            .orElseThrow(() -> new ChannelBindingNotFoundException(
                request.getStoreId(), request.getChannelType()
            ));

        // 4. 创建渠道订单
        ChannelOrder channelOrder = ChannelOrder.create(
            binding,
            request.getExternalOrderNo(),
            request.getRawOrderData()
        );

        // 5. 保存订单
        channelOrderRepository.save(channelOrder);

        // 6. 发布事件
        eventPublisher.publish(channelOrder.getDomainEvents());

        // 7. 异步解析订单
        asyncParseOrder(channelOrder.getChannelOrderId());

        return ReceiveChannelOrderResponse.success();
    }

    /**
     * 解析渠道订单
     */
    public void parseChannelOrder(String channelOrderId) {
        ChannelOrder channelOrder = channelOrderRepository.findById(channelOrderId)
            .orElseThrow(() -> new ChannelOrderNotFoundException(channelOrderId));

        try {
            // 解析订单
            UnifiedOrderPayload payload = parserService.parse(channelOrder);

            // 更新状态
            channelOrder.parse(payload);
            channelOrderRepository.save(channelOrder);

            // 创建内部订单
            createInternalOrder(channelOrder);

        } catch (Exception e) {
            log.error("解析渠道订单失败: {}", channelOrderId, e);
            channelOrder.markFailed("PARSE_ERROR", e.getMessage());
            channelOrderRepository.save(channelOrder);
        }
    }

    /**
     * 创建内部订单
     */
    private void createInternalOrder(ChannelOrder channelOrder) {
        try {
            UnifiedOrderPayload payload = channelOrder.getUnifiedOrderPayload();

            CreateOrderRequest orderRequest = CreateOrderRequest.builder()
                .storeId(channelOrder.getStoreId())
                .channelType(channelOrder.getChannelType())
                .orderScene(payload.getOrderScene())
                .fulfillmentType(payload.getFulfillmentType())
                .customerId(payload.getCustomerInfo().getCustomerId())
                .customerName(payload.getCustomerInfo().getCustomerName())
                .customerPhone(payload.getCustomerInfo().getCustomerPhone())
                .items(payload.getItems())
                .totalAmount(payload.getTotalAmount())
                .discountAmount(payload.getDiscountAmount())
                .payableAmount(payload.getPayableAmount())
                .serviceContext(payload.getServiceContext())
                .externalOrderNo(channelOrder.getExternalOrderNo())
                .build();

            CreateOrderResponse response = orderApplicationService.createOrder(orderRequest);

            channelOrder.markOrderCreated(response.getOrderId());
            channelOrderRepository.save(channelOrder);

            eventPublisher.publish(channelOrder.getDomainEvents());

        } catch (Exception e) {
            log.error("创建内部订单失败: {}", channelOrder.getChannelOrderId(), e);
            channelOrder.markFailed("CREATE_ORDER_ERROR", e.getMessage());
            channelOrderRepository.save(channelOrder);
        }
    }

    /**
     * 重试失败的渠道订单
     */
    @Scheduled(fixedDelay = 300000) // 每5分钟
    public void retryFailedOrders() {
        List<ChannelOrder> failedOrders = channelOrderRepository.findRetryableOrders(3);

        for (ChannelOrder order : failedOrders) {
            try {
                order.markRetryPending();
                channelOrderRepository.save(order);
                parseChannelOrder(order.getChannelOrderId());
            } catch (Exception e) {

        }

        } catch (Exception e) {
            log.error("异步解析订单失败: {}", channelOrderId, e);
        }
    }
```

```java
@Service
public class ChannelRateLimiterService {

    @Autowired
    private RedisTemplate<String, String> redisTemplate;

    /**
     * 检查渠道API调用频率
     * 美团限制：100次/秒
     * 饿了么限制：50次/秒
     */
    public boolean tryAcquire(ChannelType channelType, String storeId) {
        String key = "channel:rate:" + channelType.name() + ":" + storeId;
        int limit = getLimit(channelType);

        Long count = redisTemplate.opsForValue().increment(key);
        if (count == 1) {
            redisTemplate.expire(key, 1, TimeUnit.SECONDS);
        }

        return count <= limit;
    }

    private int getLimit(ChannelType channelType) {
        switch (channelType) {
            case MEITUAN:
                return 100;
            case ELEME:
                return 50;
            default:
                return 200;
        }
    }
}
```

## 15. 渠道API限流与退避策略增强

### 15.1 现有限流的不足

第13.2节的基础限流仅做了计数拦截，缺少：
- 被限流时的退避等待策略（直接抛异常会导致任务丢失）
- 渠道返回429/限流错误码时的处理
- 不同操作类型的差异化限流（查询 vs 写操作）

### 15.2 带退避的限流执行器

```java
@Service
public class ChannelApiExecutor {

    private static final Map<ChannelType, RateLimiter> RATE_LIMITERS = Map.of(
        ChannelType.MEITUAN, RateLimiter.create(80),  // 留20%余量
        ChannelType.ELEME,   RateLimiter.create(40)
    );

    /**
     * 执行渠道API调用，内置限流+指数退避重试
     * @param maxRetries 最大重试次数（建议3次）
     */
    public <T> T execute(ChannelType channel, String operation,
                         Supplier<T> apiCall, int maxRetries) {
        // 令牌桶限流：超限时最多等待500ms，否则走降级
        RateLimiter limiter = RATE_LIMITERS.get(channel);
        if (!limiter.tryAcquire(500, TimeUnit.MILLISECONDS)) {
            // 限流降级：写入待重试队列，不直接失败
            retryQueue.enqueue(channel, operation, apiCall);
            throw new ChannelRateLimitException(channel, "已入队等待重试");
        }

        Exception lastEx = null;
        for (int attempt = 0; attempt <= maxRetries; attempt++) {
            try {
                return apiCall.get();
            } catch (ChannelApiRateLimitException e) {
                // 渠道返回限流错误（429）：指数退避
                lastEx = e;
                long waitMs = (long) (Math.pow(2, attempt) * 1000) + ThreadLocalRandom.current().nextInt(500);
                log.warn("渠道限流 channel={} op={} attempt={} waitMs={}", channel, operation, attempt, waitMs);
                sleepQuietly(waitMs);
            } catch (ChannelApiException e) {
                // 非限流错误：区分可重试 vs 不可重试
                if (!e.isRetryable()) throw e;
                lastEx = e;
                long waitMs = (long) (Math.pow(2, attempt) * 500);
                sleepQuietly(waitMs);
            }
        }
        throw new ChannelApiMaxRetryException(channel, operation, maxRetries, lastEx);
    }

    private void sleepQuietly(long ms) {
        try { Thread.sleep(ms); } catch (InterruptedException e) { Thread.currentThread().interrupt(); }
    }
}
```

### 15.4 商品同步冲突解决

渠道拒绝商品同步时的处理矩阵：

| 拒绝原因 | 处理策略 |
|---|---|
| 商品名称含违禁词 | 记录冲突，告警运营人员手动修改 |
| 图片不符合规范 | 标记该商品为`SYNC_BLOCKED`，跳过同步，不影响其他商品 |
| 价格低于平台最低价 | 记录冲突，使用渠道最低价覆盖（可配置） |
| 渠道商品ID不存在 | 重新创建渠道商品，更新映射关系 |
| 渠道店铺已关闭 | 暂停该店铺所有同步任务，告警 |

```java
public SyncResult syncProduct(ChannelProductSyncCommand cmd) {
    try {
        channelApiExecutor.execute(cmd.getChannelType(), "PRODUCT_SYNC",
            () -> channelAdapter.pushProduct(cmd), 3);
        return SyncResult.success();
    } catch (ChannelApiException e) {
        ConflictResolution resolution = conflictResolver.resolve(e, cmd);
        switch (resolution.getAction()) {
            case BLOCK:
                productMappingRepository.markSyncBlocked(cmd.getMappingId(), e.getMessage());
                alertService.notifyOperator("商品同步被拒绝", cmd, e);
                return SyncResult.blocked(e.getMessage());
            case OVERRIDE_AND_RETRY:
                return syncProduct(resolution.getModifiedCommand());
            case RECREATE:
                channelProductService.deleteAndRecreate(cmd);
                return SyncResult.recreated();
            default:
                throw e;
        }
    }
}
```

### 15.5 渠道状态与内部状态的差异检测

定时任务（每15分钟）主动拉取渠道订单状态，与内部状态对比：

```java
@Scheduled(fixedDelay = 900_000)
public void detectStatusDivergence() {
    // 查找过去2小时内处于中间状态的渠道订单
    List<ChannelOrder> inFlightOrders = channelOrderRepository
        .findInFlightOrders(LocalDateTime.now().minusHours(2));

    for (ChannelOrder order : inFlightOrders) {
        ChannelOrderStatus channelStatus = channelAdapter.queryOrderStatus(order);
        if (channelStatus.isDivergentFrom(order.getInternalStatus())) {
            // 差异写入异常台账，由运营治理域处理
            exceptionRecordService.createDivergenceRecord(order, channelStatus);
        }
    }
}
```
