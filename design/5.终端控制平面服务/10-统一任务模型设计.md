# 10-统一任务模型设计

## 1. 统一任务模型概述

### 1.1 设计目标

将配置发布、热更新、远程控制三套分裂的任务模型统一为一个任务模型，实现：

- **统一的任务生命周期**：所有任务类型共享相同的状态机
- **统一的投递机制**：通过 TDP 统一投递
- **统一的结果追踪**：通过 TerminalTaskInstance 追踪每个终端的执行状态
- **差异化的 Payload**：不同任务类型通过 payload 区分业务逻辑

### 1.2 核心概念

```
TaskRelease（任务发布）
  ├─ 任务类型：CONFIG_PUBLISH / APP_UPGRADE / REMOTE_CONTROL
  ├─ 目标圈选：orgScopeId + profileScopeId + terminalIds
  ├─ 任务 Payload：不同类型有不同的 payload 结构
  └─ 生成 → TerminalTaskInstance（终端任务实例）
       ├─ 每个终端一个实例
       ├─ 统一状态机：PENDING → DISPATCHED → DELIVERED → EXECUTING → SUCCESS/FAILED
       └─ 执行结果：TaskResult（JSON）
```

## 2. TaskRelease 聚合根

### 2.1 领域模型

```java
@Entity
@Table(name = "task_releases")
public class TaskRelease {

    @Id
    private String releaseId;  // 任务发布唯一标识

    private String releaseName;  // 任务名称

    @Enumerated(EnumType.STRING)
    private TaskType taskType;  // 任务类型

    // 任务 Payload（不同类型有不同结构）
    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private TaskPayload payload;

    // 目标范围（已解算）
    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private TargetScope targetScope;

    // 灰度策略
    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private GrayscaleStrategy grayscaleStrategy;

    // 任务状态
    @Enumerated(EnumType.STRING)
    private ReleaseStatus status;

    // 优先级
    @Enumerated(EnumType.STRING)
    private TaskPriority priority;

    // 时间信息
    private LocalDateTime scheduledAt;  // 计划执行时间
    private LocalDateTime startedAt;    // 实际开始时间
    private LocalDateTime completedAt;  // 完成时间
    private LocalDateTime expiresAt;    // 过期时间

    // 统计信息
    private Integer totalCount;      // 总终端数
    private Integer dispatchedCount; // 已投递数
    private Integer deliveredCount;  // 已送达数
    private Integer executingCount;  // 执行中数
    private Integer successCount;    // 成功数
    private Integer failedCount;     // 失败数

    // 审计信息
    private String createdBy;
    private LocalDateTime createdAt;
    private LocalDateTime updatedAt;

    // 版本号（乐观锁）
    @Version
    private Integer version;

    /**
     * 创建任务发布
     */
    public static TaskRelease create(CreateTaskReleaseCommand command) {
        // 1. 参数校验
        validateCommand(command);

        // 2. 创建任务发布
        TaskRelease release = new TaskRelease();
        release.releaseId = IdGenerator.generateReleaseId();
        release.releaseName = command.getReleaseName();
        release.taskType = command.getTaskType();
        release.payload = command.getPayload();
        release.targetScope = command.getTargetScope();
        release.grayscaleStrategy = command.getGrayscaleStrategy();
        release.status = ReleaseStatus.PENDING;
        release.priority = command.getPriority() != null ?
            command.getPriority() : TaskPriority.NORMAL;
        release.scheduledAt = command.getScheduledAt() != null ?
            command.getScheduledAt() : LocalDateTime.now();
        release.expiresAt = command.getExpiresAt();
        release.createdBy = command.getCreatedBy();
        release.createdAt = LocalDateTime.now();
        release.updatedAt = LocalDateTime.now();

        // 3. 发布领域事件
        release.addDomainEvent(new TaskReleaseCreatedEvent(release));

        return release;
    }

    /**
     * 开始执行
     */
    public void start() {
        if (this.status != ReleaseStatus.PENDING) {
            throw new IllegalStateException(
                "只能启动待执行状态的任务，当前状态：" + this.status
            );
        }

        // 检查是否过期
        if (this.expiresAt != null && LocalDateTime.now().isAfter(this.expiresAt)) {
            throw new IllegalStateException("任务已过期，无法执行");
        }

        this.status = ReleaseStatus.IN_PROGRESS;
        this.startedAt = LocalDateTime.now();
        this.updatedAt = LocalDateTime.now();

        this.addDomainEvent(new TaskReleaseStartedEvent(this));
    }

    /**
     * 完成任务
     */
    public void complete() {
        if (this.status != ReleaseStatus.IN_PROGRESS) {
            throw new IllegalStateException(
                "只能完成执行中状态的任务，当前状态：" + this.status
            );
        }

        this.status = ReleaseStatus.COMPLETED;
        this.completedAt = LocalDateTime.now();
        this.updatedAt = LocalDateTime.now();

        this.addDomainEvent(new TaskReleaseCompletedEvent(this));
    }

    /**
     * 任务失败
     */
    public void fail(String reason) {
        if (this.status == ReleaseStatus.COMPLETED) {
            throw new IllegalStateException("已完成的任务不能标记为失败");
        }

        this.status = ReleaseStatus.FAILED;
        this.completedAt = LocalDateTime.now();
        this.updatedAt = LocalDateTime.now();

        this.addDomainEvent(new TaskReleaseFailedEvent(this, reason));
    }

    /**
     * 更新统计
     */
    public void updateStats(TaskStats stats) {
        this.totalCount = stats.getTotalCount();
        this.dispatchedCount = stats.getDispatchedCount();
        this.deliveredCount = stats.getDeliveredCount();
        this.executingCount = stats.getExecutingCount();
        this.successCount = stats.getSuccessCount();
        this.failedCount = stats.getFailedCount();
        this.updatedAt = LocalDateTime.now();

        // 检查是否全部完成
        int finishedCount = this.successCount + this.failedCount;
        if (finishedCount == this.totalCount && this.status == ReleaseStatus.IN_PROGRESS) {
            if (this.failedCount == 0) {
                this.complete();
            } else if (this.successCount == 0) {
                this.fail("所有终端执行失败");
            } else {
                this.complete();
            }
        }
    }

    /**
     * 计算成功率
     */
    public double getSuccessRate() {
        if (totalCount == null || totalCount == 0) {
            return 0.0;
        }
        return (double) successCount / totalCount * 100;
    }

    private static void validateCommand(CreateTaskReleaseCommand command) {
        if (command.getReleaseName() == null || command.getReleaseName().isEmpty()) {
            throw new IllegalArgumentException("任务名称不能为空");
        }
        if (command.getTaskType() == null) {
            throw new IllegalArgumentException("任务类型不能为空");
        }
        if (command.getPayload() == null) {
            throw new IllegalArgumentException("任务 Payload 不能为空");
        }
        if (command.getTargetScope() == null) {
            throw new IllegalArgumentException("目标范围不能为空");
        }
    }
}
```

## 4. TerminalTaskInstance 实体

### 4.1 领域模型

```java
@Entity
@Table(name = "terminal_task_instances")
public class TerminalTaskInstance {

    @Id
    private String instanceId;  // 实例唯一标识

    private String releaseId;   // 关联的任务发布 ID
    private String terminalId;  // 目标终端 ID

    @Enumerated(EnumType.STRING)
    private TaskType taskType;  // 任务类型

    // 任务 Payload（冗余存储，便于查询）
    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private TaskPayload payload;

    // 任务状态（统一状态机）
    @Enumerated(EnumType.STRING)
    private TaskInstanceStatus status;

    // 时间信息
    private LocalDateTime createdAt;     // 创建时间
    private LocalDateTime dispatchedAt;  // 投递时间（TCP → TDP）
    private LocalDateTime deliveredAt;   // 送达时间（TDP → 终端）
    private LocalDateTime executingAt;   // 开始执行时间
    private LocalDateTime completedAt;   // 完成时间

    // 执行结果
    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private TaskResult result;

    // 错误信息
    private String errorCode;
    private String errorMessage;

    // 重试信息
    private Integer retryCount;
    private LocalDateTime lastRetryAt;

    /**
     * 创建终端任务实例
     */
    public static TerminalTaskInstance create(
            String releaseId,
            String terminalId,
            TaskType taskType,
            TaskPayload payload) {

        TerminalTaskInstance instance = new TerminalTaskInstance();
        instance.instanceId = IdGenerator.generateInstanceId();
        instance.releaseId = releaseId;
        instance.terminalId = terminalId;
        instance.taskType = taskType;
        instance.payload = payload;
        instance.status = TaskInstanceStatus.PENDING;
        instance.createdAt = LocalDateTime.now();
        instance.retryCount = 0;

        return instance;
    }

    /**
     * 标记为已投递
     */
    public void markDispatched() {
        if (this.status != TaskInstanceStatus.PENDING) {
            throw new IllegalStateException("只能投递待执行状态的任务");
        }

        this.status = TaskInstanceStatus.DISPATCHED;
        this.dispatchedAt = LocalDateTime.now();
    }

    /**
     * 标记为已送达
     */
    public void markDelivered() {
        if (this.status != TaskInstanceStatus.DISPATCHED) {
            throw new IllegalStateException("只能标记已投递的任务为送达");
        }

        this.status = TaskInstanceStatus.DELIVERED;
        this.deliveredAt = LocalDateTime.now();
    }

    /**
     * 标记为执行中
     */
    public void markExecuting() {
        if (this.status != TaskInstanceStatus.DELIVERED) {
            throw new IllegalStateException("只能标记已送达的任务为执行中");
        }

        this.status = TaskInstanceStatus.EXECUTING;
        this.executingAt = LocalDateTime.now();
    }

    /**
     * 标记为成功
     */
    public void markSuccess(TaskResult result) {
        if (this.status == TaskInstanceStatus.SUCCESS) {
            throw new IllegalStateException("任务已标记为成功");
        }

        this.status = TaskInstanceStatus.SUCCESS;
        this.completedAt = LocalDateTime.now();
        this.result = result;
        this.errorCode = null;
        this.errorMessage = null;
    }

    /**
     * 标记为失败
     */
    public void markFailed(String errorCode, String errorMessage) {
        if (this.status == TaskInstanceStatus.SUCCESS) {
            throw new IllegalStateException("成功的任务不能标记为失败");
        }

        this.status = TaskInstanceStatus.FAILED;
        this.completedAt = LocalDateTime.now();
        this.errorCode = errorCode;
        this.errorMessage = errorMessage;
    }

    /**
     * 重试
     */
    public void retry() {
        if (this.status == TaskInstanceStatus.SUCCESS) {
            throw new IllegalStateException("成功的任务不能重试");
        }

        this.status = TaskInstanceStatus.PENDING;
        this.retryCount++;
        this.lastRetryAt = LocalDateTime.now();
        this.errorCode = null;
        this.errorMessage = null;
    }

    /**
     * 是否可以重试
     */
    public boolean canRetry(int maxRetryCount) {
        return this.status == TaskInstanceStatus.FAILED &&
               this.retryCount < maxRetryCount;
    }
}

/**
 * 终端任务实例状态（统一状态机）
 */
public enum TaskInstanceStatus {
    PENDING("待投递", "任务已创建，等待投递到 TDP"),
    DISPATCHED("已投递", "任务已投递到 TDP，等待送达终端"),
    DELIVERED("已送达", "任务已送达终端，等待执行"),
    EXECUTING("执行中", "终端正在执行任务"),
    SUCCESS("成功", "任务执行成功"),
    FAILED("失败", "任务执行失败");

    private final String displayName;
    private final String description;

    TaskInstanceStatus(String displayName, String description) {
        this.displayName = displayName;
        this.description = description;
    }
}
```

## 6. 统一状态机

```
┌─────────┐
│ PENDING │ 任务已创建，等待投递到 TDP
└────┬────┘
     │ TCP 调用 TDP API
     ▼
┌────────────┐
│ DISPATCHED │ 任务已投递到 TDP，等待送达终端
└─────┬──────┘
      │ TDP 推送到终端（WebSocket）
      ▼
┌───────────┐
│ DELIVERED │ 任务已送达终端，等待执行
└─────┬─────┘
      │ 终端开始执行
      ▼
┌───────────┐
│ EXECUTING │ 终端正在执行任务
└─────┬─────┘
      │ 终端回报结果
      ▼
┌─────────┬─────────┐
│ SUCCESS │ FAILED  │
└─────────┴─────────┘
```

**状态说明：**

- **PENDING**：TCP 内部状态，任务已创建但未投递
- **DISPATCHED**：TCP 已调用 TDP API，TDP 已接受任务
- **DELIVERED**：TDP 已将任务推送到终端（通过 WebSocket）
- **EXECUTING**：终端已开始执行任务（可选状态，终端可选择性上报）
- **SUCCESS**：终端执行成功并回报结果
- **FAILED**：终端执行失败并回报错误

**状态权威：**

- **PENDING → DISPATCHED**：TCP 控制
- **DISPATCHED → DELIVERED**：TDP 同步给 TCP
- **DELIVERED → EXECUTING → SUCCESS/FAILED**：终端回报给 TCP

## 7. 与原设计的对比

### 7.1 原设计问题

**分裂的模型：**
- ConfigRelease + ConfigTaskInstance
- UpgradeRelease + UpgradeTaskInstance
- ControlTask + TaskExecutionRecord

**问题：**
- 三套独立的聚合根，代码重复
- 三套独立的状态机，逻辑不一致
- 三套独立的 TDP 接口，协议复杂

### 7.2 统一模型优势

**统一的聚合根：**
- TaskRelease（任务发布）
- TerminalTaskInstance（终端任务实例）

**优势：**
- 代码复用，减少重复
- 统一状态机，逻辑一致
- 统一 TDP 接口，协议简化
- 易于扩展新任务类型

## 8. 本文结论

统一任务模型设计实现了配置发布、热更新、远程控制的模型统一：

- **TaskRelease**：统一的任务发布聚合根
- **TerminalTaskInstance**：统一的终端任务实例
- **TaskPayload**：差异化的任务载荷（ConfigPublishPayload / AppUpgradePayload / RemoteControlPayload）
- **TaskResult**：差异化的任务结果（ConfigPublishResult / AppUpgradeResult / RemoteControlResult）
- **统一状态机**：PENDING → DISPATCHED → DELIVERED → EXECUTING → SUCCESS/FAILED
