# 13-安全设计

## 1. 概述

TCP 服务的安全设计包括：
- **认证授权**：基于 RBAC 的权限模型
- **API 安全**：JWT Token 认证、权限校验
- **审计日志**：操作审计、数据变更追踪
- **数据保护**：敏感数据加密、软删除

### 3.2 Token 生成

```java
/**
 * JWT Token 生成器
 */
@Component
public class JwtTokenGenerator {

    @Value("${tcp.jwt.secret}")
    private String jwtSecret;

    @Value("${tcp.jwt.expiration:604800}")  // 默认7天
    private long jwtExpiration;

    /**
     * 生成终端 Token
     */
    public JwtTokenPair generateTerminalToken(TerminalInstance terminal) {
        // 获取终端角色的权限
        Set<SystemPermission> permissions = rolePermissionMapping
            .getPermissions(SystemRole.TERMINAL);

        // 构建 Claims
        Map<String, Object> claims = new HashMap<>();
        claims.put("sub", terminal.getTerminalId());
        claims.put("role", SystemRole.TERMINAL.name());
        claims.put("permissions", permissions.stream()
            .map(SystemPermission::getCode)
            .collect(Collectors.toList()));
        claims.put("tenantId", terminal.getTenantId());
        claims.put("storeId", terminal.getStoreId());

        // 生成 Access Token
        String accessToken = Jwts.builder()
            .setClaims(claims)
            .setIssuedAt(new Date())
            .setExpiration(new Date(System.currentTimeMillis() + jwtExpiration * 1000))
            .signWith(SignatureAlgorithm.HS512, jwtSecret)
            .compact();

        // 生成 Refresh Token（有效期30天）
        String refreshToken = Jwts.builder()
            .setSubject(terminal.getTerminalId())
            .setIssuedAt(new Date())
            .setExpiration(new Date(System.currentTimeMillis() + 30 * 24 * 3600 * 1000))
            .signWith(SignatureAlgorithm.HS512, jwtSecret)
            .compact();

        return JwtTokenPair.builder()
            .accessToken(accessToken)
            .refreshToken(refreshToken)
            .expiresAt(LocalDateTime.now().plusSeconds(jwtExpiration))
            .build();
    }

    /**
     * 生成管理员 Token
     */
    public String generateAdminToken(String userId, SystemRole role, String tenantId) {
        // 获取角色的权限
        Set<SystemPermission> permissions = rolePermissionMapping
            .getPermissions(role);

        // 构建 Claims
        Map<String, Object> claims = new HashMap<>();
        claims.put("sub", userId);
        claims.put("role", role.name());
        claims.put("permissions", permissions.stream()
            .map(SystemPermission::getCode)
            .collect(Collectors.toList()));
        claims.put("tenantId", tenantId);

        // 生成 Token
        return Jwts.builder()
            .setClaims(claims)
            .setIssuedAt(new Date())
            .setExpiration(new Date(System.currentTimeMillis() + jwtExpiration * 1000))
            .signWith(SignatureAlgorithm.HS512, jwtSecret)
            .compact();
    }
}
```

### 3.3 Token 解析

```java
/**
 * JWT Token 解析器
 */
@Component
public class JwtTokenParser {

    @Value("${tcp.jwt.secret}")
    private String jwtSecret;

    /**
     * 解析 Token
     */
    public TcpTokenClaims parseToken(String token) {
        try {
            Claims claims = Jwts.parser()
                .setSigningKey(jwtSecret)
                .parseClaimsJws(token)
                .getBody();

            TcpTokenClaims tokenClaims = new TcpTokenClaims();
            tokenClaims.setSub(claims.getSubject());
            tokenClaims.setRole(claims.get("role", String.class));
            tokenClaims.setPermissions(claims.get("permissions", List.class));
            tokenClaims.setTenantId(claims.get("tenantId", String.class));
            tokenClaims.setStoreId(claims.get("storeId", String.class));
            tokenClaims.setExp(claims.getExpiration().getTime() / 1000);
            tokenClaims.setIat(claims.getIssuedAt().getTime() / 1000);

            return tokenClaims;

        } catch (ExpiredJwtException e) {
            throw new TokenExpiredException("Token 已过期");
        } catch (JwtException e) {
            throw new InvalidTokenException("Token 无效");
        }
    }

    /**
     * 验证 Token
     */
    public boolean validateToken(String token) {
        try {
            Jwts.parser().setSigningKey(jwtSecret).parseClaimsJws(token);
            return true;
        } catch (JwtException e) {
            return false;
        }
    }
}
```

## 4. API 权限校验

### 4.1 权限注解

```java
/**
 * 权限校验注解
 */
@Target({ElementType.METHOD, ElementType.TYPE})
@Retention(RetentionPolicy.RUNTIME)
public @interface RequirePermission {
    SystemPermission[] value();
    LogicalOperator operator() default LogicalOperator.AND;
}

/**
 * 逻辑运算符
 */
public enum LogicalOperator {
    AND,  // 需要所有权限
    OR    // 需要任一权限
}
```

### 4.2 权限拦截器

```java
/**
 * 权限拦截器
 */
@Component
@Aspect
public class PermissionInterceptor {

    @Autowired
    private JwtTokenParser jwtTokenParser;

    @Around("@annotation(requirePermission)")
    public Object checkPermission(ProceedingJoinPoint joinPoint, RequirePermission requirePermission)
            throws Throwable {

        // 1. 从请求头获取 Token
        HttpServletRequest request = ((ServletRequestAttributes)
            RequestContextHolder.currentRequestAttributes()).getRequest();

        String authHeader = request.getHeader("Authorization");
        if (authHeader == null || !authHeader.startsWith("Bearer ")) {
            throw new UnauthorizedException("缺少认证信息");
        }

        String token = authHeader.substring(7);

        // 2. 解析 Token
        TcpTokenClaims claims = jwtTokenParser.parseToken(token);

        // 3. 检查权限
        SystemPermission[] requiredPermissions = requirePermission.value();
        List<String> userPermissions = claims.getPermissions();

        boolean hasPermission;
        if (requirePermission.operator() == LogicalOperator.AND) {
            // 需要所有权限
            hasPermission = Arrays.stream(requiredPermissions)
                .allMatch(p -> userPermissions.contains(p.getCode()));
        } else {
            // 需要任一权限
            hasPermission = Arrays.stream(requiredPermissions)
                .anyMatch(p -> userPermissions.contains(p.getCode()));
        }

        if (!hasPermission) {
            throw new ForbiddenException("权限不足");
        }

        // 4. 将 Claims 放入 ThreadLocal，供后续使用
        SecurityContextHolder.setContext(claims);

        try {
            return joinPoint.proceed();
        } finally {
            SecurityContextHolder.clear();
        }
    }
}
```

### 5.2 终端安全拦截器

```java
/**
 * 终端安全拦截器
 */
@Component
@Aspect
public class TerminalSecurityInterceptor {

    @Around("@annotation(terminalSecurity)")
    public Object checkTerminalAccess(ProceedingJoinPoint joinPoint, TerminalSecurity terminalSecurity)
            throws Throwable {

        // 1. 获取当前认证信息
        TcpTokenClaims claims = SecurityContextHolder.getContext();

        // 2. 如果不是终端角色，跳过检查（管理员可以访问所有终端）
        if (!SystemRole.TERMINAL.name().equals(claims.getRole())) {
            return joinPoint.proceed();
        }

        // 3. 获取请求中的 terminalId
        String requestTerminalId = extractTerminalId(joinPoint, terminalSecurity.terminalIdParam());

        // 4. 校验终端只能访问自己的数据
        if (!claims.getSub().equals(requestTerminalId)) {
            throw new ForbiddenException("无权访问其他终端的数据");
        }

        return joinPoint.proceed();
    }

    private String extractTerminalId(ProceedingJoinPoint joinPoint, String paramName) {
        // 从方法参数中提取 terminalId
        MethodSignature signature = (MethodSignature) joinPoint.getSignature();
        String[] parameterNames = signature.getParameterNames();
        Object[] args = joinPoint.getArgs();

        for (int i = 0; i < parameterNames.length; i++) {
            if (parameterNames[i].equals(paramName)) {
                return (String) args[i];
            }
        }

        throw new IllegalArgumentException("未找到参数：" + paramName);
    }
}
```

```java
/**
 * 审计日志

@Entity
@Table(name = "audit_logs")
public class AuditLog {

    @Id
    private String auditId;

    // 操作信息
    @Enumerated(EnumType.STRING)
    private OperationType operationType;  // CREATE, UPDATE, DELETE, EXECUTE

    private String entityType;  // TerminalInstance, TaskRelease, ConfigVersion
    private String entityId;

    // 操作前后数据
    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private Map<String, Object> beforeData;

    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private Map<String, Object> afterData;

    // 操作人信息
    private String operator;      // 操作人ID（userId 或 terminalId）
    private String operatorRole;  // 操作人角色
    private String operatorIp;    // 操作人IP

    // 时间信息
    private LocalDateTime operatedAt;

    // 操作结果
    @Enumerated(EnumType.STRING)
    private OperationResult result;  // SUCCESS, FAILURE

    private String errorMessage;  // 失败原因

    /**
     * 创建审计日志
     */
    public static AuditLog create(
            OperationType operationType,
            String entityType,
            String entityId,
            Map<String, Object> beforeData,
            Map<String, Object> afterData,
            String operator,
            String operatorRole,
            String operatorIp) {

        AuditLog log = new AuditLog();
        log.auditId = IdGenerator.generateAuditId();
        log.operationType = operationType;
        log.entityType = entityType;
        log.entityId = entityId;
        log.beforeData = beforeData;
        log.afterData = afterData;
        log.operator = operator;
        log.operatorRole = operatorRole;
        log.operatorIp = operatorIp;
        log.operatedAt = LocalDateTime.now();
        log.result = OperationResult.SUCCESS;

        return log;
    }

    /**
     * 标记失败
     */
    public void markFailed(String errorMessage) {
        this.result = OperationResult.FAILURE;
        this.errorMessage = errorMessage;
    }
}

 * 操作类型

    CREATE,   // 创建
    UPDATE,   // 更新
    DELETE,   // 删除
    EXECUTE   // 执行
}

 * 操作结果
```

### 6.2 审计日志拦截器

```java
/**
 * 审计日志拦截器
 */
@Component
@Aspect
public class AuditLogInterceptor {

    @Autowired
    private AuditLogRepository auditLogRepository;

    @Around("@annotation(auditable)")
    public Object audit(ProceedingJoinPoint joinPoint, Auditable auditable) throws Throwable {
        // 1. 获取当前认证信息
        TcpTokenClaims claims = SecurityContextHolder.getContext();
        HttpServletRequest request = ((ServletRequestAttributes)
            RequestContextHolder.currentRequestAttributes()).getRequest();

        // 2. 记录操作前数据
        Map<String, Object> beforeData = extractBeforeData(joinPoint, auditable);

        // 3. 执行操作
        Object result;
        AuditLog auditLog = null;
        try {
            result = joinPoint.proceed();

            // 4. 记录操作后数据
            Map<String, Object> afterData = extractAfterData(result, auditable);

            // 5. 创建审计日志
            auditLog = AuditLog.create(
                auditable.operationType(),
                auditable.entityType(),
                extractEntityId(joinPoint, result),
                beforeData,
                afterData,
                claims.getSub(),
                claims.getRole(),
                request.getRemoteAddr()
            );

        } catch (Exception e) {
            // 记录失败日志
            auditLog = AuditLog.create(
                auditable.operationType(),
                auditable.entityType(),
                extractEntityId(joinPoint, null),
                beforeData,
                null,
                claims.getSub(),
                claims.getRole(),
                request.getRemoteAddr()
            );
            auditLog.markFailed(e.getMessage());

            throw e;
        } finally {
            if (auditLog != null) {
                auditLogRepository.save(auditLog);
            }
        }

        return result;
    }
}

/**
 * 审计注解
 */
@Target({ElementType.METHOD})
@Retention(RetentionPolicy.RUNTIME)
public @interface Auditable {
    OperationType operationType();
    String entityType();
}
```

```java
/**
 * 审计日志查询服务
 */
@Service
public class AuditLogQueryService {

    @Autowired
    private AuditLogRepository auditLogRepository;

    /**
     * 查询审计日志
     */
    public PageResult<AuditLogDTO> queryAuditLogs(AuditLogQuery query) {
        // 构建查询条件
        Specification<AuditLog> spec = (root, criteriaQuery, criteriaBuilder) -> {
            List<Predicate> predicates = new ArrayList<>();

            // 按实体类型过滤
            if (query.getEntityType() != null) {
                predicates.add(criteriaBuilder.equal(
                    root.get("entityType"), query.getEntityType()));
            }

            // 按实体ID过滤
            if (query.getEntityId() != null) {
                predicates.add(criteriaBuilder.equal(
                    root.get("entityId"), query.getEntityId()));
            }

            // 按操作人过滤
            if (query.getOperator() != null) {
                predicates.add(criteriaBuilder.equal(
                    root.get("operator"), query.getOperator()));
            }

            // 按时间范围过滤
            if (query.getStartTime() != null) {
                predicates.add(criteriaBuilder.greaterThanOrEqualTo(
                    root.get("operatedAt"), query.getStartTime()));
            }
            if (query.getEndTime() != null) {
                predicates.add(criteriaBuilder.lessThanOrEqualTo(
                    root.get("operatedAt"), query.getEndTime()));
            }

            return criteriaBuilder.and(predicates.toArray(new Predicate[0]));
        };

        // 执行查询
        Page<AuditLog> page = auditLogRepository.findAll(
            spec,
            PageRequest.of(
                query.getPage() - 1,
                query.getSize(),
                Sort.by(Sort.Direction.DESC, "operatedAt")
            )
        );

        // 转换为 DTO
        return PageResult.of(
            page.getContent().stream()
                .map(this::toDTO)
                .collect(Collectors.toList()),
            page.getTotalElements(),
            query.getPage(),
            query.getSize()
        );
    }

    /**
     * 查询实体变更历史
     */
    public List<AuditLogDTO> getEntityHistory(String entityType, String entityId) {
        List<AuditLog> logs = auditLogRepository
            .findByEntityTypeAndEntityIdOrderByOperatedAtDesc(entityType, entityId);

        return logs.stream()
            .map(this::toDTO)
            .collect(Collectors.toList());
    }
}
```

### 6.5 审计日志归档

**归档策略：**
- 保留最近90天的审计日志在主表
- 90天以上的日志归档到历史表
- 历史表保留2年

```sql
-- 创建审计日志历史表
CREATE TABLE audit_logs_history (
    LIKE audit_logs INCLUDING ALL
);

-- 归档脚本（每天执行）
INSERT INTO audit_logs_history
SELECT * FROM audit_logs
WHERE operated_at < NOW() - INTERVAL '90 days';

DELETE FROM audit_logs
WHERE operated_at < NOW() - INTERVAL '90 days';
```

## 7. 本文结论

安全设计展示了完整的认证授权机制：

- **RBAC 权限模型**：定义角色（PLATFORM_ADMIN, TENANT_ADMIN, STORE_MANAGER, TERMINAL）和权限
- **JWT Token**：包含角色、权限、租户信息，有效期7天
- **API 权限校验**：使用 `@RequirePermission` 注解进行权限校验
- **终端访问控制**：终端只能访问自己的数据，结果回报以 Token 中的 terminalId 为准
- **审计日志**：记录所有关键操作，包括操作前后数据、操作人、操作结果

所有 API 都有明确的权限要求，终端 API 有额外的访问范围限制。
