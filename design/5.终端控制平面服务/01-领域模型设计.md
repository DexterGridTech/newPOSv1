# 01-领域模型设计

## 1. 领域模型概述

TCP 服务的领域模型围绕终端资产管理和运维控制展开，核心聚合根包括：

- **TerminalInstance**（终端实例）：真实的终端设备
- **DeviceProfile**（设备能力模板）：终端能力定义
- **ActivationCode**（激活码）：终端激活凭证
- **TaskRelease**（任务发布）：统一的任务发布模型（配置/升级/远控）
- **TerminalTaskInstance**（终端任务实例）：终端级任务执行实例

**注意**：配置管理、热更新、远程控制已统一为 TaskRelease 模型，详见 [10-统一任务模型设计](./10-统一任务模型设计.md)。

### 1.1 并发控制策略

TCP 服务使用多层并发控制机制：

| 场景 | 并发控制策略 | 实现方式 |
|------|-------------|---------|
| 激活码消费 | 分布式锁 | Redis 分布式锁（lockKey: `activation:lock:{code}`） |
| 终端换绑 | 悲观锁 | JPA `@Lock(PESSIMISTIC_WRITE)` |
| 任务结果回报 | 乐观锁 + 重试 | JPA `@Version` + `@Retryable(maxAttempts=3)` |
| 任务发布 | 分布式锁 | Redis 分布式锁（lockKey: `task:release:lock:{releaseId}`） |
| 终端状态更新 | 乐观锁 | JPA `@Version` |

**详细设计**：参见 [12-分布式事务与一致性设计.md](./12-分布式事务与一致性设计.md)

## 2. TerminalInstance 聚合根

### 2.1 领域模型

```java
@Entity
@Table(name = "terminal_instances")
public class TerminalInstance {

    @Id
    private String terminalId;

    // 设备信息
    private String deviceId;
    private String deviceSn;
    private String deviceFingerprint;

    // 组织归属
    private String projectId;
    private String tenantId;
    private String brandId;
    private String storeId;

    // 模板与配置
    private String profileId;
    private String templateId;

    // 状态信息（三类状态分层）
    @Enumerated(EnumType.STRING)
    private LifecycleStatus lifecycleStatus;  // 生命周期状态（TCP权威）

    @Enumerated(EnumType.STRING)
    private PresenceStatus presenceStatus;  // 在线状态（TDP同步）

    @Enumerated(EnumType.STRING)
    private HealthStatus healthStatus;  // 健康状态（TCP计算）

    // 版本信息
    private String currentAppVersion;
    private String currentBundleVersion;
    private String currentConfigVersion;

    // 时间信息
    private LocalDateTime installedAt;
    private LocalDateTime activatedAt;
    private LocalDateTime lastSeenAt;
    private LocalDateTime retiredAt;

    // 标签
    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private List<String> tags;

    // 能力
    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")
    private List<String> capabilities;

    // 版本号（乐观锁）
    @Version
    private Integer version;

    private LocalDateTime createdAt;
    private LocalDateTime updatedAt;

    // 领域方法

    /**
     * 创建终端实例
     */
    public static TerminalInstance create(CreateTerminalCommand command) {
        TerminalInstance terminal = new TerminalInstance();
        terminal.terminalId = IdGenerator.generateTerminalId();
        terminal.deviceId = command.getDeviceId();
        terminal.deviceSn = command.getDeviceSn();
        terminal.deviceFingerprint = command.getDeviceFingerprint();
        terminal.projectId = command.getProjectId();
        terminal.tenantId = command.getTenantId();
        terminal.brandId = command.getBrandId();
        terminal.storeId = command.getStoreId();
        terminal.profileId = command.getProfileId();
        terminal.templateId = command.getTemplateId();
        terminal.lifecycleStatus = LifecycleStatus.DRAFT;
        terminal.presenceStatus = PresenceStatus.UNKNOWN;
        terminal.healthStatus = HealthStatus.HEALTHY;
        terminal.tags = command.getTags();
        terminal.capabilities = command.getCapabilities();
        terminal.createdAt = LocalDateTime.now();
        terminal.updatedAt = LocalDateTime.now();

        // 发布领域事件
        terminal.addDomainEvent(new TerminalCreatedEvent(terminal));

        return terminal;
    }

    /**
     * 激活终端
     */
    public void activate(ActivationCode activationCode) {
        if (this.lifecycleStatus != LifecycleStatus.PENDING_ACTIVATION) {
            throw new IllegalStateException("终端状态不正确，无法激活");
        }

        this.lifecycleStatus = LifecycleStatus.ACTIVATED;
        this.activatedAt = LocalDateTime.now();
        this.updatedAt = LocalDateTime.now();

        // 发布领域事件
        this.addDomainEvent(new TerminalActivatedEvent(this, activationCode));
    }

    /**
     * 绑定终端
     */
    public void bind(String storeId, String profileId, String templateId) {
        if (this.lifecycleStatus != LifecycleStatus.ACTIVATED) {
            throw new IllegalStateException("终端状态不正确，无法绑定");
        }

        this.storeId = storeId;
        this.profileId = profileId;
        this.templateId = templateId;
        this.lifecycleStatus = LifecycleStatus.BOUND;
        this.updatedAt = LocalDateTime.now();

        // 发布领域事件
        this.addDomainEvent(new TerminalBoundEvent(this));
    }

    /**
     * 换绑终端
     */
    public void rebind(String newStoreId, String newProfileId, String newTemplateId, String reason) {
        if (this.lifecycleStatus != LifecycleStatus.BOUND) {
            throw new IllegalStateException("终端状态不正确，无法换绑");
        }

        String oldStoreId = this.storeId;
        String oldProfileId = this.profileId;
        String oldTemplateId = this.templateId;

        this.storeId = newStoreId;
        this.profileId = newProfileId;
        this.templateId = newTemplateId;
        this.updatedAt = LocalDateTime.now();

        // 发布领域事件
        this.addDomainEvent(new TerminalReboundEvent(
            this, oldStoreId, oldProfileId, oldTemplateId, reason));
    }

    /**
     * 暂停终端
     */
    public void suspend(String reason) {
        if (this.lifecycleStatus == LifecycleStatus.RETIRED) {
            throw new IllegalStateException("已退役的终端无法暂停");
        }

        this.lifecycleStatus = LifecycleStatus.SUSPENDED;
        this.updatedAt = LocalDateTime.now();

        // 发布领域事件
        this.addDomainEvent(new TerminalSuspendedEvent(this, reason));
    }

    /**
     * 恢复终端
     */
    public void resume() {
        if (this.lifecycleStatus != LifecycleStatus.SUSPENDED) {
            throw new IllegalStateException("终端状态不正确，无法恢复");
        }

        this.lifecycleStatus = LifecycleStatus.BOUND;
        this.updatedAt = LocalDateTime.now();

        // 发布领域事件
        this.addDomainEvent(new TerminalResumedEvent(this));
    }

    /**
     * 退役终端
     */
    public void retire(String reason) {
        if (this.lifecycleStatus == LifecycleStatus.RETIRED) {
            throw new IllegalStateException("终端已退役");
        }

        this.lifecycleStatus = LifecycleStatus.RETIRED;
        this.retiredAt = LocalDateTime.now();
        this.updatedAt = LocalDateTime.now();

        // 发布领域事件
        this.addDomainEvent(new TerminalRetiredEvent(this, reason));
    }

    /**
     * 更新在线状态（由 TDP 同步）
     */
    public void updatePresenceStatus(PresenceStatus newStatus) {
        if (this.presenceStatus != newStatus) {
            this.presenceStatus = newStatus;
            this.lastSeenAt = LocalDateTime.now();
            this.updatedAt = LocalDateTime.now();

            // 发布领域事件
            this.addDomainEvent(new TerminalPresenceStatusChangedEvent(this, newStatus));
        }
    }

    /**
     * 更新版本信息
     */
    public void updateVersion(String appVersion, String bundleVersion, String configVersion) {
        this.currentAppVersion = appVersion;
        this.currentBundleVersion = bundleVersion;
        this.currentConfigVersion = configVersion;
        this.updatedAt = LocalDateTime.now();

        // 发布领域事件
        this.addDomainEvent(new TerminalVersionUpdatedEvent(this));
    }

    /**
     * 更新健康状态
     */
    public void updateHealthStatus(HealthStatus newStatus, String reason) {
        if (this.healthStatus != newStatus) {
            this.healthStatus = newStatus;
            this.updatedAt = LocalDateTime.now();

            // 发布领域事件
            this.addDomainEvent(new TerminalHealthStatusChangedEvent(this, newStatus, reason));
        }
    }

    /**
     * 添加标签
     */
    public void addTag(String tag) {
        if (this.tags == null) {
            this.tags = new ArrayList<>();
        }
        if (!this.tags.contains(tag)) {
            this.tags.add(tag);
            this.updatedAt = LocalDateTime.now();
        }
    }

    /**
     * 移除标签
     */
    public void removeTag(String tag) {
        if (this.tags != null && this.tags.contains(tag)) {
            this.tags.remove(tag);
            this.updatedAt = LocalDateTime.now();
        }
    }
}
```

### 2.2 值对象

```java
/**
 * 生命周期状态（TCP 权威）
 */
public enum LifecycleStatus {
    DRAFT("草稿", "终端记录已创建但未激活"),
    PENDING_ACTIVATION("待激活", "等待激活码激活"),
    ACTIVATED("已激活", "已激活但未绑定门店"),
    BOUND("已绑定", "已绑定门店，正常使用"),
    SUSPENDED("暂停使用", "暂时停用"),
    RETIRED("已退役", "永久退役");

    private final String displayName;
    private final String description;

    LifecycleStatus(String displayName, String description) {
        this.displayName = displayName;
        this.description = description;
    }
}

/**
 * 在线状态（TDP 同步）
 */
public enum PresenceStatus {
    ONLINE("在线", "终端已连接到 TDP"),
    OFFLINE("离线", "终端未连接到 TDP"),
    UNKNOWN("未知", "尚未建立过连接");

    private final String displayName;
    private final String description;

    PresenceStatus(String displayName, String description) {
        this.displayName = displayName;
        this.description = description;
    }
}

/**
 * 健康状态（TCP 计算）
 */
public enum HealthStatus {
    HEALTHY("健康", "终端运行正常"),
    WARNING("警告", "终端有潜在问题"),
    ERROR("错误", "终端存在严重问题");

    private final String displayName;
    private final String description;

    HealthStatus(String displayName, String description) {
        this.displayName = displayName;
        this.description = description;
    }
}
```

**三类状态说明：**

1. **LifecycleStatus（生命周期状态）**
   - **权威来源**：TCP 控制面
   - **变更方式**：通过 TCP API 操作（激活、绑定、暂停、退役等）
   - **用途**：控制终端的业务生命周期

2. **PresenceStatus（在线状态）**
   - **权威来源**：TDP 数据面
   - **变更方式**：TDP 通过 WebHook 同步给 TCP
   - **用途**：反映终端与 TDP 的连接状态

3. **HealthStatus（健康状态）**
   - **权威来源**：TCP 计算
   - **变更方式**：TCP 根据多种指标综合计算（版本过期、长时间离线、任务失败率高等）
   - **用途**：运维监控和告警

@Entity
@Table(name = "device_profiles")
public class DeviceProfile {

    @Id
    private String profileId;

    private String profileName;
    private String profileCode;

    @Enumerated(EnumType.STRING)
    private DeviceCategory deviceCategory;  // POS, KDS, PDA, KIOSK, DISPLAY, APP

    private String deviceModel;
    private String platform;  // android-rn84, electron, ios, web

    // 能力列表
    @Type(JsonBinaryType.class)
    @Column(columnDefinition = "jsonb")

    // 配置 Schema 集合 ID
    private String configSchemaSetId;

    @Enumerated(EnumType.STRING)
    private ProfileStatus status;  // ACTIVE, DEPRECATED

    private LocalDateTime createdAt;
    private LocalDateTime updatedAt;

    // 领域方法

    /**
     * 创建设备能力模板
     */
    public static DeviceProfile create(CreateDeviceProfileCommand command) {
        DeviceProfile profile = new DeviceProfile();
        profile.profileId = IdGenerator.generateProfileId();
        profile.profileName = command.getProfileName();
        profile.profileCode = command.getProfileCode();
        profile.deviceCategory = command.getDeviceCategory();
        profile.deviceModel = command.getDeviceModel();
        profile.platform = command.getPlatform();
        profile.capabilities = command.getCapabilities();
        profile.allowedRemoteCommands = command.getAllowedRemoteCommands();
        profile.supportedUpgradeModes = command.getSupportedUpgradeModes();
        profile.configSchemaSetId = command.getConfigSchemaSetId();
        profile.status = ProfileStatus.ACTIVE;
        profile.createdAt = LocalDateTime.now();
        profile.updatedAt = LocalDateTime.now();

        // 发布领域事件
        profile.addDomainEvent(new DeviceProfileCreatedEvent(profile));

        return profile;
    }

    /**
     * 废弃设备能力模板
     */
    public void deprecate() {
        this.status = ProfileStatus.DEPRECATED;
        this.updatedAt = LocalDateTime.now();

        // 发布领域事件
        this.addDomainEvent(new DeviceProfileDeprecatedEvent(this));
    }
}

## 4. ActivationCode 聚合根

```java
@Entity
@Table(name = "activation_codes")
public class ActivationCode {

    @Id
    private String activationCodeId;

    private String code;
    private String batchId;

    @Enumerated(EnumType.STRING)
    private ActivationCodeStatus status;  // NEW, USED, EXPIRED, REVOKED

    // 组织归属
    private String projectId;
    private String tenantId;
    private String brandId;
    private String storeId;

    // 模板
    private String profileId;
    private String templateId;

    // 使用限制
    private Integer maxUseCount;
    private Integer usedCount;
    private LocalDateTime expiresAt;

    // 创建信息
    private String issuedBy;
    private LocalDateTime createdAt;
    private LocalDateTime updatedAt;

    // 领域方法

    /**
     * 创建激活码
     */
    public static ActivationCode create(CreateActivationCodeCommand command) {
        ActivationCode code = new ActivationCode();
        code.activationCodeId = IdGenerator.generateActivationCodeId();
        code.code = ActivationCodeGenerator.generate();
        code.batchId = command.getBatchId();
        code.status = ActivationCodeStatus.NEW;
        code.projectId = command.getProjectId();
        code.tenantId = command.getTenantId();
        code.brandId = command.getBrandId();
        code.storeId = command.getStoreId();
        code.profileId = command.getProfileId();
        code.templateId = command.getTemplateId();
        code.maxUseCount = command.getMaxUseCount();
        code.usedCount = 0;
        code.expiresAt = command.getExpiresAt();
        code.issuedBy = command.getIssuedBy();
        code.createdAt = LocalDateTime.now();
        code.updatedAt = LocalDateTime.now();

        // 发布领域事件
        code.addDomainEvent(new ActivationCodeCreatedEvent(code));

        return code;
    }

    /**
     * 使用激活码
     */
    public void use(String terminalId) {
        // 校验状态
        if (this.status != ActivationCodeStatus.NEW) {
            throw new IllegalStateException("激活码状态不正确，无法使用");
        }

        // 校验有效期
        if (this.expiresAt != null && LocalDateTime.now().isAfter(this.expiresAt)) {
            this.status = ActivationCodeStatus.EXPIRED;
            throw new IllegalStateException("激活码已过期");
        }

        // 校验使用次数
        if (this.usedCount >= this.maxUseCount) {
            throw new IllegalStateException("激活码使用次数已达上限");
        }

        // 增加使用次数
        this.usedCount++;
        if (this.usedCount >= this.maxUseCount) {
            this.status = ActivationCodeStatus.USED;
        }
        this.updatedAt = LocalDateTime.now();

        // 发布领域事件
        this.addDomainEvent(new ActivationCodeUsedEvent(this, terminalId));
    }

    /**
     * 撤销激活码
     */
    public void revoke(String reason) {
        if (this.status == ActivationCodeStatus.REVOKED) {
            throw new IllegalStateException("激活码已撤销");
        }

        this.status = ActivationCodeStatus.REVOKED;
        this.updatedAt = LocalDateTime.now();

        // 发布领域事件
        this.addDomainEvent(new ActivationCodeRevokedEvent(this, reason));
    }

    /**
     * 检查是否可用
     */
    public boolean isAvailable() {
        if (this.status != ActivationCodeStatus.NEW) {
            return false;
        }
        if (this.expiresAt != null && LocalDateTime.now().isAfter(this.expiresAt)) {
            return false;
        }
        if (this.usedCount >= this.maxUseCount) {
            return false;
        }
        return true;
    }
}

/**
 * 激活码状态
 */
public enum ActivationCodeStatus {
    NEW,      // 新建
    USED,     // 已使用
    EXPIRED,  // 已过期
    REVOKED   // 已撤销
}
```

## 5. 本文结论

TCP 服务的领域模型设计展示了完整的终端资产管理实现：

- **TerminalInstance**：终端实例聚合根，包含完整的生命周期管理方法
  - 三类状态分层：LifecycleStatus（TCP权威）、PresenceStatus（TDP同步）、HealthStatus（TCP计算）
- **DeviceProfile**：设备能力模板，定义终端能力和支持的操作
- **ActivationCode**：激活码聚合根，支持激活、使用次数控制、撤销等操作

所有聚合根都包含：
- 完整的领域方法实现
- 领域事件发布
- 状态机管理
- 乐观锁支持

**任务模型**：配置管理、热更新、远程控制已统一为 TaskRelease 模型，详见 [10-统一任务模型设计](./10-统一任务模型设计.md)。
